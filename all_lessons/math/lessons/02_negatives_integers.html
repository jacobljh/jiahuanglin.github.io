减法的裂缝：负数与整数

自然数对加法、乘法都封闭，是个安稳的小世界。可一道最普通的减法 3 − 5，结果却跑出了这个世界。要堵上这道裂缝，我们被迫造出一种被人类抵制了三百年的数——负数。

线性回顾
上一课：我们发明了证明与数学归纳法，能用有限的几句话锁死关于无穷多自然数的断言（高斯求和、素数无穷）。我们终于"会谈论"整个自然数世界了。
留下的问题：自然数 ℕ 对加法、乘法封闭——算完不会跑出去。但减法不是这样：3 − 5 = ? 在自然数里压根没有答案。减法这扇门，一脚就把我们踹出了 ℕ。
本课新增：负数与整数 ℤ，让减法处处可行；并要弄清一件几百年没人想通的事——"负负得正"为什么必须如此。
本课路线
(1) 减法的封闭性裂缝：3 − 5 无处落脚；(2) 负数被抵制三百年的历史；(3) 数轴的发明——把负数安放成"方向"；(4) 关键洞见：负数的运算规则不是规定，是被分配律逼出来的，推导 (−1)×(−1) = +1；(5) 整数 ℤ 对加减乘封闭。

一道踹开门的减法

上一课我们夸自然数"慷慨"：任意两个相加、相乘，结果还是自然数，从不跑出去——这叫封闭。加法封闭，乘法封闭，看起来 ℕ 是个自给自足的小世界。

但减法是加法的"反着问"：3 − 5 等于"几加上 5 才得 3"。在自然数里，加上 5 只会越走越大，永远回不到 3。于是：

3 − 5 = ?　在 ℕ 里：无解（找不到任何自然数能填进这个空）

这就是一道封闭性裂缝：减法把我们带到了自然数的边界之外，那里空无一物。你有两个选择——要么宣布"3 − 5 这种式子非法、禁止去问"，要么把数轴向左延长，造出新的数来接住它。数学史一次次证明：每当工具撞墙，"造新东西"才是出路。但这一次，出路走得格外艰难。

被抵制了三百年的数

今天小学生都用负数，但它曾被欧洲最聪明的头脑斥为"荒谬"。十六、十七世纪的数学家把负数叫作"虚构的数""比无还少的怪物"，帕斯卡公开说"0 减 4 等于 0，因为没有什么能比无更少"。一个方程要是解出负根，常被直接当作"无意义"丢掉。直到十七、十八世纪，负数才在欧洲被普遍接受。

耐人寻味的是，更早的印度和中国数学家反而用得自如——因为他们把负数接到了现实：正数记"盈余/收入"，负数记"欠债/支出"；正数记"温度在零上"，负数记"零下"。−5 不是"比无还少的鬼怪"，而是"欠了 5 块钱"，清清楚楚。这给了我们第一条线索：负数难以接受，不是因为它不真实，而是因为当时缺一张能"看见"它的图。那张图，就是数轴。

数轴：把"负"翻译成"方向"

把所有数排在一条水平直线上：取一点记作 0，向右一步步是 1, 2, 3, …，再把这条线向左也延长，对称地标上 −1, −2, −3, …。这一笔下去，负数立刻有了身体：

… −3　−2　−1　0　1　2　3 …　（越往左越小，越往右越大）

在这张图上，减法变成了一个干脆利落的动作——向左走。算 3 − 5：从 3 出发，向左走 5 步，落在 0 左边的 −2 上。门一下子就通了。而且数轴顺手澄清了一个常见的糊涂账：比大小看的是谁更靠右，不是看数字"长得大不大"。−2 在 −5 的右边，所以 −2 > −5，尽管 5 比 2 "大"。下面的小实验，就让一个小人在数轴上替你走这趟减法。

数轴行走者：让减法向左走
设定起点 a 和要减去的 b，点"走"，看小人从 a 出发向左走 b 步，落在哪里。当终点掉到 0 左边，就说明这道减法在自然数里无解、只有整数能接住它。切到第二个模式，还能看 (−1)×(−1) 为什么必须是 +1。

"负负得正"不是规定，是被逼的

很多人记得老师那句"负负得正"，却觉得它像一条天降的法令。其实它根本不是谁规定的——它是被一条我们舍不得放弃的老规律逼出来的必然结果。这条老规律就是分配律：

a × (b + c) = a × b + a × c　（乘法可以"分着"作用到加法的每一项上）

分配律在自然数里天经地义（3 个"2+5"就是 3 个 2 加 3 个 5）。当我们把数系扩张到负数，有一个底线要求：原来的运算规律必须继续成立，否则新数系就是另起炉灶、和旧世界对不上。守住分配律，(−1)×(−1) 的值就由不得我们挑了。看这条推理：

先看一个一定为 0 的式子：　1 + (−1) = 0

两边同乘 (−1)：　(−1) × [ 1 + (−1) ] = (−1) × 0 = 0

用分配律拆开左边：　(−1)×1 + (−1)×(−1) = 0

而 (−1)×1 = −1，代入：　(−1) + (−1)×(−1) = 0

读到这里，关键的一步来了：式子说"−1 加上某个东西等于 0"，那个东西只能是 +1（因为 −1 的相反数就是 +1，没有别的数加 −1 能得 0）。所以：

(−1) × (−1) = +1　（这是被分配律逼出来的，不是约定）

这正是本课最该带走的洞见：扩张数系时，新数的运算规则不能随便定，必须由"保留旧规律"来唯一确定。负负得正之所以是这样，是因为任何别的取值都会让分配律崩掉，把整座算术大厦拆了。负数不是凭空硬塞的，它是被旧规律"接纳"进来的，且只能以这一种方式被接纳。

整数 ℤ：减法终于无处可逃

把自然数、它们的负数、再加上 0，凑成一个新集合：

ℤ = { …, −3, −2, −1, 0, 1, 2, 3, … }　（读作整数集合，向两边无限延伸）

在 ℤ 里，任意两个整数相加、相减、相乘，结果都还是整数——它对加、减、乘三种运算全都封闭。上一课那道踹开门的 3 − 5，如今稳稳落在 −2 上，再没有"无解"的尴尬。我们用一次扩张，把减法的裂缝彻底堵死了。

可是别急着庆祝。回头看 ℤ 的三种封闭运算，名单上少了一个：除法。这正是下一道墙的位置。

常见误解

  • "负数根本不存在、没有意义。"
    历史上确实有人这么想三百年。但欠债、零下温度、海平面以下的深度、账户透支……负数描述的是方向相反的同一种量，和正数一样真实。数轴把这份真实画了出来。
  • "负负得正只是老师的规定，背下来就行。"
    恰恰相反，它是被分配律逼出来的唯一结果。想保留 a×(b+c)=ab+ac 这条老规律，(−1)×(−1) 就只能等于 +1，换任何别的值都会让算术自相矛盾。
  • "−5 比 −2 大，因为 5 比 2 大。"
    比大小看的是数轴上谁更靠右，不是数字"长得大不大"。−5 在 −2 的左边，所以 −5 < −2。欠 5 块比欠 2 块更穷，对应的数也更小。
一句话带走
减法把我们踹出了自然数，于是我们向数轴左侧造出负数、凑成整数 ℤ，让加减乘处处可行；而"负负得正"不是规定，是"保留分配律"逼出的唯一选择——扩张数系，从来都得让旧规律说了算。
