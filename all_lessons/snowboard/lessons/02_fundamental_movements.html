第一部分 · CASI 基本功（把工具箱吃透）

四类基本动作：转体 / 屈伸 / 给刃 / 前后配重

上一课把底座立稳了：现在你站在一个"随时能发力、板前后都能响应"的位置上，四个旋钮都够得着。可"够得着"不等于"会拧"。这一课就正式认识那四只手本身——它们各自到底在调「刃 × 压力」的哪一部分？CASI 把你能对板做的一切，干净地归成四类基本动作（fundamental movements）：转体、纵向屈伸、横向给刃、前后配重。看懂一件事你就赚到了整门课：这四类，就是你调「刃 × 压力」的全部手段——没有第五种；之后每一个进阶动作（刻线、动态转弯、switch、ollie、跳台、转体），都只是这四类在不同任务下的重新组合。顺带，把那个和"后座"并列的头号病灶点破：用手臂甩着带转身——转体的力，从来不在手臂。

本课路线
(1) 先立方法论：为什么"四类"就够了——把「刃 × 压力」拆成"往哪走、弧多弯、抓不抓得住、压在哪一段"四个问题，每个问题恰好对应一类动作；(2) 逐一推导四类动作，每类都从"要调引擎的哪一部分"反推出身体该怎么动——① 转体 Rotation（定指向，核心+下肢带板转）、② 纵向屈伸 Flexion/Extension（定压力大小、吸地形）、③ 横向给刃 Tilt/Edging（定刃角）、④ 前后配重 Fore–aft（定压力沿板的分布）；(3) 重点点破"甩手臂转身"为什么是死路，正确的转体从哪里发起；(4) 动手：一个个拨动这四类动作，亲眼看板如何分别响应——转向、压力、刃角、配重各自变什么；最后交棒第 03 课——认全了四只手，就该看它们如何按转弯三相协同成一个完整的弧。

一、为什么恰好是"四类"：把引擎拆成四个问题

上一课我们说，滑行只在调"刃×压力"两个旋钮。但"调"这个动作，其实藏着四个独立的问题——把这两个旋钮摊开细看：

问题 A：板往哪个方向走？板此刻指向哪里、要不要转向。这决定了弧朝哪边开。
问题 B：弧要多弯？转得急还是缓。这由刃立得多陡决定——刃角越大弧越紧。
问题 C：刃抓得牢不牢、地形颠不颠得住？压上去的力有多大。力足，刃咬得实、颠簸吸得住；力松，刃就打滑。
问题 D：这些力压在板的哪一段？压在板头还是板尾。这微调弧的形状、决定板头板尾谁先咬雪。

四个问题，各自独立、又缺一不可。妙就妙在——身体恰好有四类动作，一一对应地回答这四个问题：

① 转体 Rotation → 回答 A（指向）用核心与下肢把板转向要去的方向。
③ 横向给刃 Tilt / Edging → 回答 B（弧多弯）把板从平放倾倒到刃上，决定刃角。
② 纵向屈伸 Flexion / Extension → 回答 C（压力大小）腿的弯伸决定压上多少力、也吸收地形。
④ 前后配重 Fore–aft → 回答 D（压力分布）重心沿板前后移动，决定力压在板头还是板尾。

这就是为什么 CASI 把基本动作定为这四类——不是随便凑的清单，而是把"管理刃×压力"这件事做完整、所必需且足够的四个自由度。往哪走、多弯、多牢、压哪段——问全了，一个弧的所有变量就都在你手里了。

先记住这张对应表
转体＝定指向（往哪走）；给刃＝定刃角（弧多弯）；屈伸＝定压力大小（牢不牢、吸不吸震）；前后配重＝定压力分布（压板头还是板尾）。四类动作 = 调「刃×压力」的四个旋钮方向，没有第五种。你之后学的每个花哨动作，都只是这四类的一次特定组合与排序。

二、① 转体 Rotation：定指向——但力不在手臂

任务：让板转向你要去的方向（问题 A）。物理：板要转，需要一个绕竖轴的转动；这个转动得从一个有质量、又稳的地方发起，并且要能持续地、可控地传给板。于是动作被推出来：转动应从核心（躯干）与下肢（髋、腿、脚踝）发起——由大肌肉带动，经腿脚把"转"的意图传到板上，让整块板顺着转。

这就引出全线第二个头号病灶（第一个是后座）——用手臂 / 肩膀甩着带转身。新手一想转弯，本能就是把上半身、手臂往目标方向一甩。为什么这是死路？

手臂太轻、又在末端手臂质量小、离转轴远，甩它产生的有效转动传不到板上；你把上半身甩过去了，板却没跟着转——力断在了半路。这正是全库那条动力链引擎反复讲的病：用末端小肌肉去凑，跳过了核心和下肢这些大肌肉。
甩手破坏平衡与刃、压力上半身猛地一扭，重心跟着晃、肩髋错位——你上一课辛苦堆好的堆叠塌了，刃立不实、压力送错地方。想转的没转成，稳的先丢了。而且甩过去之后无处可去，只能反向"收"回来，动作又顿又乱。
正确的转体从哪里发起
想着用核心和前髋"引导"板去你眼睛看的方向（记得上一课"眼看去处"？转体正是让身体兑现眼睛的指令）。下肢——尤其前腿的髋和踝——把这个引导传给板，让整块板一起转。手臂只做一件事：放松、张开、帮你维持平衡与节奏，绝不是发动机。一个检验：转弯时如果你觉得是"肩膀在带头、手在划圈"，八成错了；应该是"肚子和腿在带板走、肩自然跟随"。

三、③ 横向给刃 Tilt / Edging：定刃角——弧有多弯

任务：决定弧转得多急（问题 B）。物理：上一课讲过，板立起刃、加上侧切板型(sidecut)的弧度，就被迫沿一条弧走；刃角越大，那条弧越紧（这背后的侧切物理，第 05 课会细讲）。于是动作被推出来：你需要一个把板从平放倾倒到刃上的横向动作——这就是给刃(edging / tilt)。

给刃靠什么做？主要是踝、膝、髋的横向倾倒（angulation）与身体的适度内倾（inclination）配合：踝在靴中压向脚趾或脚跟（前刃 / 后刃）、膝和髋朝山里侧倒，让板立起。给多少刃，就得多弯的弧。它是"方向盘的转向幅度"——转体决定你打不打方向盘、往哪打，给刃决定你打多少。

前刃（toe edge）与后刃（heel edge）单板有两条刃——脚趾侧和脚跟侧。一个完整转弯往往是从一条刃换到另一条刃。给刃这个动作，就是在这两条刃之间切换、并控制各自的角度。
刃角小 vs 刃角大刃角小：板接近平放，容易侧滑刮雪（skid）、弧很缓甚至走直。刃角大：板深切进雪，弧紧、走得干净——这是刻线(carve)的方向。第 04 课就专讲从小刃角的"刮"走到大刃角的"刻"。

四、② 纵向屈伸 Flexion / Extension：定压力大小——牢不牢、吸不吸震

任务：决定压上板的力有多大，以及吸收地形起伏（问题 C）。物理：压力＝让刃真正"咬"进雪的那个力；力大，刃抓得牢、弧稳；同时，地形一颠，你需要一个"缓冲行程"把冲击吃掉，否则会被弹飞。于是动作被推出来：你需要腿的屈曲（flexion，弯腿下沉）与伸展（extension，蹬腿升高）——这就是纵向屈伸。它正是上一课那把"上了膛的弹簧"在真正做功。

1伸展（extension）加压 / 屈曲（flexion）减压——或反过来，取决于时序。最直观的理解：像你站在体重秤上，猛地伸腿蹬地，读数会瞬间飙高（加压）；猛地屈腿下坠，读数会瞬间变小甚至失重（减压）。你就是这样用腿的伸屈，主动地把压力调大调小，喂给刃。（加压到底该配伸展还是屈曲，取决于是 cross-over 还是 cross-under——那是第 06 课的动态转弯。）
2吸收地形（absorption）。遇到包、坎、硬雪，主动屈腿把冲击"吞"进关节的行程里，让板贴住雪、让上半身保持安静——而不是被硬生生弹起来失去抓地。这就是为什么上一课要留屈曲余量：余量就是吸收的行程。
3屈伸是"压力控制"的主力。整条线里凡提到"管理压力峰值""吸收起伏""load 板蓄力"，主角都是屈伸。ollie 的爆发下压反弹、动态转弯的压力时机、跳台落地的主动吸收——全是屈伸在不同任务下的用法。

五、④ 前后配重 Fore–aft：定压力分布——压在板头还是板尾

任务：决定这些压力压在板的哪一段（问题 D）。物理：同样的总压力，压在板头还是板尾，效果完全不同——发起弯时压力略向板头去，能让板头先咬雪、帮你把弧"带"起来；弧到后段可把压力收回中间或略后，稳住收尾。于是动作被推出来：你需要把重心沿板前后微移——这就是前后配重（fore–aft）。上一课那个"重心前后位置"滑块，调的正是这一类动作。

它和上一课的关系很直接：上一课教你把重心居中当作原点（别死在一头，尤其别后座）；这一课告诉你，从这个居中原点出发、有意识地向前或向后微调压力，本身就是四类基本动作之一。后座之所以毁一切，正是因为它让你失去了前后配重的能力——重心焊死在尾，压力再也调不动分布。配重是"精调"，前提是你有一个能自由前后的中正底座。

四类动作串起来看
一个漂亮的弯里，四类动作是协同发生的：眼睛看向出口 → 转体用核心+下肢把板引向那里（指向）→ 同时给刃把板立起到需要的刃角（弧多弯）→ 屈伸把压力喂上去让刃咬雪、并吸收地形（多牢）→ 前后配重把压力在板头板尾间调到最合适（压哪段）。四只手一起拧，「刃×压力」就被你完整地管起来了。它们怎么排时序，就是下一课的"转弯三相"。

六、动手：一个个拨动四类动作，看板如何响应

四类动作各管一件事，最好的理解方式是把它们分开、一个一个地拨，看板分别有什么反应。下面这块俯视 + 示意面板让你做到这点：用四个滑块分别控制转体、给刃、屈伸、前后配重，画面里那块板会实时响应——转体改指向、给刃改弧的弯度、屈伸改压力大小（板边"咬雪"的深浅）、配重改压力压在板头还是板尾。右侧读数把每类动作此刻在调引擎的哪一部分写清楚。

建议先只动一个、其余归零，把每类动作的"专属效果"看孤立了：只给刃 → 弧变弯但不转向；只转体 → 指向变但没立刃（会打滑）；只屈伸 → 压力大小变、抓地深浅变；只配重 → 压力在板头板尾间挪。看清楚每只手管什么，再一起拨，体会它们如何合成一个受控的弧。

把这一课接回主线

一句话带走
你调「刃×压力」的全部手段，只有四类基本动作：转体定指向（核心+下肢带板转，不是甩手臂）、给刃定刃角（弧多弯）、屈伸定压力大小（牢不牢、吸不吸震）、前后配重定压力分布（压板头还是板尾）。它们恰好回答"往哪走、多弯、多牢、压哪段"四个问题，把一个弧的所有变量补齐。之后每个进阶动作，都是这四类在特定任务下的重新组合与排序。

四只手认全了、也知道各管什么了。可它们在一个真实的弯里不是同时"哐"地全用上，而是有先后、有节奏地登场：先松刃发起、再持续给刃引导、最后把弧收完控速。这个"什么时候用哪只手"的时序，就是把四类动作组织成一个完整弧的骨架——转弯三相。这正是下一课。

常见误解

  • 误解：转弯主要靠转肩、甩手臂把身体带过去。 （澄清：手臂太轻、又在末端，甩它转动传不到板上，还会破坏堆叠、送错刃与压力——这是"用末端小肌肉凑"的典型病。转体应由核心 + 下肢（尤其前髋、前腿踝）发起，手臂只用来平衡与节奏。）
  • 误解：给刃就是把板"扭"向一边。 （澄清：给刃是横向把板从平放倾倒到刃上（靠踝膝髋倾倒 + 身体内倾），决定的是刃角 = 弧的弯度；转向由转体负责。两者是不同的两只手，别混为一谈。）
  • 误解：屈伸（弯腿蹬腿）只是为了"蹲低"或省力。 （澄清：屈伸是压力控制的主力——伸屈直接把压上刃的力调大调小（像在体重秤上蹬腿/下坠），还负责吸收地形让板贴雪。它是你喂给刃的"力量阀门"，不是姿势装饰。）
  • 误解：前后配重和上一课的"重心居中"是矛盾的。 （澄清：不矛盾。居中是原点（别死在一头、尤其别后座）；前后配重是从这个原点出发的有意识微调（发起时略向板头、收尾回中）。后座之所以糟，正因为它让你失去了前后配重这份能力。）
  • 误解：四类动作要在转弯里同时、平均地一起用。 （澄清：它们是协同但有时序的——按转弯三相有先后登场（下一课）。而且不同动作在不同弯里权重不同：刻线偏重给刃+屈伸，短弯偏重转体+快速换刃。是"组合与排序"，不是平均堆砌。）
  • 误解：进阶要学的是几十个新招式。 （澄清：进阶动作几乎都是这四类的重新组合——刻线=给刃+屈伸推满、ollie=屈伸爆发、转体=转体放大。练熟四只手 + 会按需组合，比背招式管用得多，也正是本门课的方法论。）
