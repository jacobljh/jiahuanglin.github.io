单板滑雪 · 进阶技术 + 公园自由式 · CASI 体系

单板滑雪的技术

一条线性思维的推理线：单板滑雪只做一件事——用身体的基本动作管理「板刃(edge) × 压力(pressure)」，把重力沿 fall line（滚落线）的下坠，驯成一条条你说了算的弧。CASI（加拿大单板滑雪教练体系）把这套整理成一个工具箱：站姿与平衡（底座）+ 四类基本动作（转体 / 屈伸 / 给刃 / 前后配重）+ 转弯三相。而从刻线到公园跳台，每一个进阶动作，都只是这套工具的重新组合。

为什么是这门课
你已经能连起基本转弯，却卡在"一快就刷、一陡就慌"，也不知道怎么真正刻线、怎么上公园。这门课面向想把技术练精、进阶到刻线与自由式的你——不是零基础"别摔倒"教程。我们只立一个引擎（管理「刃 × 压力」），围绕 CASI 的基本动作框架，按「先吃透工具箱 → 把转弯从刮练到刻 → 把同一套动作搬进公园」的顺序，一课一课把每个动作推导出来。你不会背到一堆招式，而会理解每个动作为什么非这样不可——于是进阶更快、更稳、更省力、更安全。
同源 · 个人技术运动
这门课与库里几门个人技术运动同脉络——尤其《高尔夫的逻辑》：都是"你对抗的不是对手，而是重力/地形/自己的动作"，胜负系于对身体动作与物理的精确控制。它也和「从零学会打」系列（网球、足球 等各自的第 0 部分）共用同一套「从任务反推动作、力/平衡从物理逼出技术」的方法论。每课配一个浏览器里当场绘制的可交互小实验（无需任何图片或视频）。CASI 术语如实使用，技术描述以正确为先；具体数值均标「示意 / 量级」。

第一部分 · CASI 基本功（把工具箱吃透）

00
导览：单板只有一件事——管理「刃 × 压力」
用"一快就刷、一陡就慌"当钩子，立起全程引擎：单板 = 用基本动作管理「刃 × 压力」，把 fall line 的下坠驯成受控的弧。摊开 CASI 工具箱（站姿平衡 + 四类基本动作 + 转弯三相），并立起方法论：每个进阶动作都只是这套工具的重新组合。拨"给刃"滑块，看同一条坡从直冲变成刻弧。
01
站姿与平衡：CASI 的参考对位，与后座为何毁一切
四类基本动作要能做出来，前提是站在一个"随时能发力、前后都能响应"的位置。从这个任务反推出中正、屈曲、关节堆叠、视线看向去处的参考对位；再把"一快就往后坐"的本能拆开，看后座如何同时让"刃"和"压力"两个旋钮失灵。
02
四类基本动作：转体 / 屈伸 / 给刃 / 前后配重
你调"刃 × 压力"的全部手段：转体(rotation，核心+下肢带板转，不是甩手臂)、纵向屈伸(flexion/extension，管压力大小与吸收地形)、横向给刃(tilt/edging，定刃角)、前后配重(fore-aft，定压力沿板的分布)。之后每个技术都是它们的组合。
03
转弯的解剖：发起 → 控制 → 收尾三相
一个弧分三相：发起(松刃、重心过板)、控制/引导(持续给刃、转体、加压)、收尾(把弧画完整、横切 fall line 控速)。把四类基本动作按这个时序组织起来；点破"刷坡"正是跳过了收尾相，只好用侧滑硬蹭。

第二部分 · 进阶转弯（从刮雪到刻线）

04
从推刃到刻线：skid → carve
侧滑(skid)=小刃角 + 刮雪减速；刻线(carve)=大刃角 + 压力让板沿侧切自己走，板尾精准跟着板头、留下一条干净刻痕。为什么刻线更快、更稳、更省力——以及它对雪况与坡度的要求。
05
刃、反弓与侧切的物理：刃角如何决定弧的弯度
板有侧切半径(sidecut radius)；给刃 + 施压让板反弓(decamber)贴雪，有效转弯半径随刃角变小（R ≈ R侧切·cos刃角，示意）。这解释了为什么"想转更急就给更多刃"，以及板长、腰宽、硬度如何改写你的弧。
06
动态转弯：cross-over / cross-under 与压力控制
重心过板(cross-over，流畅高山)与板过重心(cross-under，快速换刃 + 吸收)是换刃的两种时序；用屈伸管理压力峰值、吸收起伏(pressure control)。同一套基本动作，改一改屈伸时机，就从优雅长弧变成敏捷短弯。
07
Switch（反脚）与对称化：自由式的地基
为什么想上公园必须先练 switch：落地、出杆、连招都要求你反脚也能滑。把 01–06 的基本动作在反脚一一复制，把身体的"惯用侧"抹平——switch 不是新技术，是把工具箱对称化。

第三部分 · 公园与自由式

08
公园安全、选线与 SMART Style
进公园的第一课是安全：Start small · Make a plan · Always look（先看再落）· Respect · Take it easy。公园是共享空间——喊出你的 drop、别停在落点下方、先小后大。怎么读一个 feature、判断速度与落点，是所有招式的前提。
09
Ollie & Nollie：用板的反弹，不是硬跳
自由式的原子：不是靠腿硬蹬，而是后脚下压 load 板尾、前脚提、在板反弹的那一刻 pop 弹起（nollie 用板头）。从"最大离地且可控"反推 load → pop → 收腿的时序——时机对了，板会"还"你一大截高度。
10
Jibbing：上盒子与杆（50-50 / boardslide）
把平衡投影稳稳落在窄面的中线上：垂直进入、锁住核心、视线看出口不看脚。50-50 是顺着走，boardslide 要用一次稳定的 90° 转体把板锁横——本质还是"给不给得住那条看不见的中线"。
11
跳台四阶段：approach → pop → flight → landing
稳定中正地进入(不后座)、在台唇 pop、空中身体安静并抓板、屈膝主动吸收、落在下坡的甜点上。全程最关键的是速度匹配台子：拍平(flat，速度不够)和过台(overshoot，速度太多)都危险——先从小台把速度感建起来。
12
转体：FS / BS 180 与 360
预转 wind-up（反向蓄力）→ 核心带动开肩起转 → spotting（提前找落点）→ 收肩控制落地(regular/switch)。frontside 与 backside 的关键差别是视野（backside 是"盲转"，更难）。转体不是甩，是把 02 的"转体"这类动作在空中放大并控住。
13
抓板与收官：把动作串成风格 + 进阶路线
抓板(indy / mute / melon / tail / nose / method)既逼你在空中收好腿、也是你的风格签名。收官回望整条链——一切都是用四类基本动作管理「刃 × 压力」；再给出进阶路线（halfpipe、大跳台、backcountry、考 CASI 等级）与安全底线。

顺带一提：这门课和它的近邻——《高尔夫的逻辑》（同为"你对抗重力/地形/自己"的个人技术运动）、以及各运动「从零学会打」的技术入门线——出自同一个家族：把"怎么把一项运动做对"讲成一条从物理反推动作的推理线，而不是一堆要背的招式。会了基本动作，进阶与公园只是它的重新组合。
