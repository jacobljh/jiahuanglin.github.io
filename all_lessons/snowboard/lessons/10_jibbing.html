第三部分 · 公园与自由式

Jibbing：上盒子与杆（50-50 / boardslide）

上一课你拿到了自由式的原子——用板的反弹，又高又稳地弹起来。现在把这一下的平衡带到一个全新的、也是最考验平衡的场景：jibbing——滑上一条又窄又滑的 box 或 rail，沿着它滑过去。你的第一反应大概是"上去以后拼命调整、别掉下来"。但那样一定会掉——因为窄面上几乎没有摩擦、没有给你刃去咬的余地。jibbing 的真相恰恰相反：它不是"上去以后再稳"，而是把平衡在进入之前就设定好，然后什么都别乱动。这一课要从一个极简的物理事实出发——要不从窄面上滑脱，你的重心投影必须始终落在那条窄面的中线上——反推出 jibbing 的全部要领：垂直进入、锁死核心、视线看出口不看脚；再看 50-50（顺着走）和 boardslide（把板锁成横的）的差别，其实只是"这条中线相对板的朝向"变了。学会读这条看不见的中线，你就读懂了整个 jibbing。

本课路线
(1) 先立起 jibbing 唯一的物理约束——重心投影 vs 窄面中线：偏出去就滑脱；(2) 从这个约束反推要领——垂直进入（别斜切上去）、锁死核心（上箱后别乱调）、视线看出口（不看脚）；(3) 讲 50-50（板顺着 box 走，最简单）与 boardslide（用一次干净的 90° 转体把板锁成横的、沿中线滑）的区别，点明本质是同一条中线；(4) 说清进入角度与速度为什么必须"正"；(5) 亲手拨一个"中线平衡"沙盘：调你的重心投影与进入角度，看是稳稳锁住还是滑脱；最后交棒第 11 课——jib 是"低空贴着 feature"的平衡，把它抬到空中、加上一个真正的 pop 与落点，就是跳台。
安全前置
jib 从最低、最宽、最平的 box 起步（宽 box 的中线"容错带"最宽），戴头盔、护腕/护臀很有用。rail（细杆）比 box 难得多、也更伤——先把宽 box 玩熟再说。落点和 box 下方别站人、别停留（回到第 08 课的共享空间规矩）。本课角度、速度、宽度都是示意 / 量级，帮你理解"重心投影必须压在中线上"这个关系，真实手感只能在低宽 box 上慢慢试。

一、jibbing 唯一的物理约束：重心投影 vs 中线

先把 jibbing 剥到只剩物理。你站在一条 box 上——它又窄（就巴掌宽）、表面又滑（塑料或金属，几乎没摩擦）。在这样的表面上，你之前赖以生存的两个旋钮几乎都失效了：你没法给刃（窄面上立刃会直接打滑或翻下去），也没有多少摩擦可用。那还剩什么能让你不掉下去？

只剩一件事：平衡。具体说，是一个极其简单、也极其无情的几何条件——

jibbing 的全部物理
把你的重心（COM）竖直往下投影到 box 上，得到一个点。只要这个点落在 box 那条窄窄的接触面（中线）上，你就稳；一旦它偏出中线，重力就会产生一个把你往那一侧翻下去的力矩，你就滑脱。就这么简单。box 越窄，容许的偏差越小（rail 几乎是一条线，所以最难）；box 越宽，中线两侧的"容错带"越宽，越好上手——这正是"先从宽 box 练"的物理根据。

想象走一条平衡木：你能过去，不是因为你在木头上不停地大幅调整，而是因为你把重心稳稳压在木头正上方、身体安静。box 一模一样。所以 jibbing 的核心任务，不是"上去以后拼命救"，而是从一开始就把重心投影放到中线上，然后尽量什么都别动——因为在没有摩擦、没有刃的窄面上，你每一次慌乱的"调整"，都极可能把重心投影甩出中线，反而害了自己。

二、从这条中线，反推出 jibbing 的三条要领

既然唯一的目标是"重心投影始终压在中线上、身体安静"，那正确的动作就被唯一地逼出来了。三条，全都是这个目标的直接推论：

1垂直进入（别斜切上去）。进入 box 时，你的动量方向要尽量顺着 box 的方向（对 50-50 就是让板正对着 box 上去）。为什么？因为如果你带着一个横向速度斜切上去，这个横向分量会立刻把你的重心投影往侧面推——还没站稳就已经偏出中线了。垂直进入，等于让你的动量一开始就"落在中线上"，不给横向偏移留机会。
2锁死核心（上箱后别乱动）。一旦上了 box，把核心收紧、身体像一根柱子一样安静地立在板正上方——不要用手臂乱划、不要扭腰、不要试图"微调"。窄面无摩擦，任何多余的动作都会晃动重心、把投影甩出中线。稳，来自"不动"，不是来自"救"。
3视线看出口，不看脚。把眼睛望向 box 的末端（出口）、望向你要去的地方，而不是低头盯着脚下的板。为什么？因为人有一个本能：身体会跟着视线走——看哪里，重心就往哪里稳定。盯着脚会让你低头、含胸、重心前倾发慌；看向远处的出口，身体自然挺直、重心稳定地沿中线往前送。这和第 01 课"视线看向要去的地方"是同一条原理，在窄面上被放大到了极致。

你看，三条要领没有一条是"新技术"——它们全是"把重心投影钉在中线上、身体安静"这一个物理目标的必然结果。垂直进入让投影一开始就在中线上；锁死核心让它别被乱动甩出去；看出口让身体自然地把它沿中线往前带。记住这条中线，你就不需要背要领——你能自己推出来。

三、50-50 与 boardslide：同一条中线，不同的板向

入门的两个 jib 动作，差别听起来很大，其实只是"这条中线相对你的板怎么摆"变了一下：

50-50：板顺着 box 走板的长边顺着 box 的方向，两只脚（"50-50"指两端都在 box 上）一起顺着滑过去。这时中线是沿着板的长轴的——你只要像平常滑行一样，把重心压在板正中、看向出口、别乱动。它是最简单的 jib，是所有人的起点：因为它对平衡的要求最接近你已经会的"直线滑行"。
boardslide：把板锁成横的用一次干净、稳定的 90° 转体，让板横过来、垂直于 box，然后横着沿 box 滑。这时中线横切过你的板中央——你的重心要压在这条横中线上，身体依然安静地立着。难点全在那次转体：它必须正好 90° 且转完就锁住（用第 02 课的转体、核心带动，不是甩手臂）；转多了、转少了、或转完还在动，重心都会偏出中线而滑脱。

把它们并排看，jibbing 的方法论就清楚了：不管板怎么摆，你要守的永远是同一件事——把重心投影压在 box 的那条中线上。50-50 里这条中线顺着板，你几乎不用改姿势；boardslide 里这条中线横切板，你要先用一次精确的 90° 转体把板摆到位、再守住。动作在变，物理目标从没变。这正是这门课反复讲的：进阶动作只是基本动作（这里是转体 + 平衡）在新任务下的重新组合。

四、进入角度与速度：为什么都要"正"

最后两个常被忽视、却决定成败的变量：进入角度和速度。

角度必须"正"。前面说过，斜切上 box 会带来横向速度、把重心投影推出中线。所以理想的进入是让你的动量方向对准 box（50-50 是板正对 box；boardslide 是让转体正好在上箱那一刻完成到横向）。进入角度越"歪"，你上箱瞬间就已经在中线之外，后面几乎救不回来——jib 的成败，一大半在你还没上箱时就定了。

速度要"够且稳"。太慢：还没滑到 box 末端就失去动量、停在中途卡住（在窄面上停住＝失衡），或者根本上不去、上去就掉。太快：来不及在上箱瞬间把姿态和板向调正、也更难在末端从容地滑出（out）。理想速度是能匀速、从容地滑完整条 box 并顺畅滑出——不快不慢，全程动量稳定。和跳台一样，这个"正好的速度"只能在低矮的宽 box 上一点点试出来。

把它接回引擎
jibbing 看起来抛开了"刃×压力"，其实是把它们逼到了极限的另一端：窄面无摩擦、不能给刃，于是你能依靠的只剩最纯粹的平衡——重心投影压在中线上。它是第 01 课"站姿与平衡"这个底座的终极考试。你管理的仍是身体这条动力链，只不过这一次，"稳定"就是全部答案，任何多余的"发力"或"调整"都是干扰。

五、动手：中线平衡沙盘（稳住还是滑脱？）

下面这块俯视的 box 让你把上面的物理亲手拨一遍。box 中间那条虚线就是中线，两侧浅色带是它的容错带（box 越宽越宽）。你能调两样东西：你的重心投影（相对中线偏左/偏右多少）和进入角度（正对 box，还是斜切上去）。右侧读数与画面会告诉你这一下是稳稳锁住还是往哪一侧滑脱，以及你偏离中线有多远。

做两个实验。其一：把进入角度保持"正"，只挪重心投影——你会看到只要它待在中央的绿色容错带里，骑手就稳稳沿 box 滑到出口（绿，"锁住"）；一旦推出容错带，重力力矩立刻把他往那侧翻下去（红，"滑脱"）。其二：把重心投影摆在正中，只把进入角度调歪——你会看到即便重心一开始在中线上，斜切带来的横向速度也会让投影一路飘出容错带、最终滑脱。这就证明了那句话：jib 的成败一大半在进入那一刻就定了——垂直进入 + 重心压中线，缺一不可。

中线平衡 · 重心投影压在中线上，才不滑脱
调 重心投影（相对 box 中线偏左/右）和 进入角度（正对／斜切）。看骑手沿中线锁住还是往一侧滑脱，以及偏离量。绿色带＝容错带（box 越宽越宽）。特意试：重心摆正、只把角度调歪——斜切的横向速度照样让你飘出中线。宽度/角度为示意值，抓"重心投影必须压在中线上""斜切会把投影推出去""越窄越难"这几个关系。

拨过之后，jibbing 的秘密就再简单不过了：它不是"上去以后救"，是"从进入那一刻起，把重心投影钉在中线上、身体安静、看向出口"。50-50 也好、boardslide 也好，招式在变，你守的永远是那条看不见的中线。把这条线读懂，你就把整个 jib 世界的门推开了。

把这一课接回主线

一句话带走
jibbing 的全部物理只有一条：你的重心投影必须始终落在 box/rail 的那条窄中线上——偏出去，重力力矩就把你翻下去。窄面无摩擦、不能给刃，于是纯平衡是唯一答案。由此唯一地推出三条要领：垂直进入（别让横向速度把投影推出中线）、锁死核心（别乱动、别乱救）、视线看出口不看脚（身体跟着视线稳）。50-50是中线顺着板（最简单），boardslide是用一次干净的 90° 转体把板锁成横的、守住横切的中线——招式在变，守的中线不变。进入角度要正、速度要够且稳，因为 jib 的成败一大半在上箱前就定了。它是第 01 课平衡底座的终极考试。

你已经能在贴着地面的窄 feature 上守住平衡了。现在把这份平衡抬到空中：如果不是贴着 box 滑，而是从一个台唇 pop 起来、在空中飞一段、再落到一个下坡上——那就是公园里最有分量、也最需要"速度匹配"的动作。下一课，我们把 ollie 的 pop、jib 的平衡、和第 08 课的速度判断，全都合进一次完整的飞行里。

常见误解

  • 误解：上了 box 之后靠不停地调整来保持平衡。 （澄清：窄面无摩擦、不能给刃，任何多余动作都会把重心投影甩出中线。稳来自"锁死核心、身体安静"，不是来自"救"。平衡要在进入之前就设定好。）
  • 误解：在 box 上要给刃、用刃咬住才稳。 （澄清：窄面上立刃会直接打滑或翻下去。jib 里你唯一能靠的是纯平衡——重心投影压在中线上，这是第 01 课站姿平衡的极限考试，和转弯时"给刃"是相反的场景。）
  • 误解：斜切着冲上 box 更容易、更顺。 （澄清：斜切带来横向速度，会立刻把重心投影推出中线，上箱瞬间就已失衡。要垂直进入、让动量对准 box——jib 的成败一大半在你还没上箱时就定了。）
  • 误解：应该低头盯着脚下的板才稳。 （澄清：身体会跟着视线走。盯脚会低头含胸、重心前倾发慌；看向 box 出口，身体自然挺直、重心稳定地沿中线前送。这是第 01 课"看向去处"在窄面上的放大。）
  • 误解：boardslide 就是随便把板扭横。 （澄清：它需要一次干净、稳定的 90° 转体（核心+下肢带动，不是甩手臂），且转完就锁住；转多、转少、或转完还在动，重心都会偏出中线而滑脱。板向变了，但守的还是那条中线。）
  • 误解：widget 里的宽度、角度、偏离量是精确数值。 （澄清：是示意 / 量级，只为让你抓住"投影必须在中线上""斜切会把投影推出去""越宽越好上手"这些关系。真实容错随 box 宽度、雪况、速度、体重差异极大，务必从最低最宽的 box 试起。）
