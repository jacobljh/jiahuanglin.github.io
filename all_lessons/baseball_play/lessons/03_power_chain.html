第二部分 · 发力与控制（把球传快、打准）

传球的动力链：抓地→跨步→转髋→鞭打

前三课，你已经把「底座」打好了：手会抓球抓棒了（01），也能站稳、能用脚先到位了（02）。可到现在为止，你还只是「稳稳地站着」——那台发动机一直没点火。这一课，我们终于要发动它：把 02 课那个不晃的稳底座真正用起来，发动整条动力链，把球又快又直地传出去。这是全课「核心的一下」——00 课我们讲了引擎的原理（力从地面来，一节节甩到手），这一课就把它落到一个具体、真实的动作上：overhand（上手）传球。你会看到，那句「力从地面来」不是口号，而是能被拆成六个可以照着做的环节。

本课路线
(1) 先把 overhand 传球的完整动力链按顺序拆成六环——抓地 → 跨步指向目标 → 髋先转 → 肩带臂 → 鞭打出手 → 随挥——每一环都从「任务→力该怎么从地面传上来→于是动作被唯一决定」推导出来；(2) 逐环讲清它为什么非这样不可、为什么是这个顺序；(3) 点破新手最大的通病——只抬肘、用手臂甩（push 而非 whip），看动力链怎么断在半路、球怎么又飘又软还伤手臂；(4) 亲手打开/关闭各个环节，眼看着「只用手臂」和「逐个接上跨步、转髋」能差出多少球速、还多省力；最后交棒第 04 课——传球的这条链你会了，挥棒用的是同一条链，只是甩的换成了棒、还多了「在对的点、对的时机触球」这道新题。

一、任务先行：这一传，你到底想要什么

动手之前，先按老规矩问一句：传球这一下，任务是什么？答案很朴素——把球又快、又直、又准地送到接球人手里，而且你的手臂还得撑得住一天。四个词：快、直、准、不伤。记住它，因为下面每一个动作，都是被这四个字逼出来的。

那力该从哪来？回到 00 课立下的引擎：不是从手臂来，是从地面来。手臂是全身最末端、最小的一组肌肉，让它当发动机，就像用一根手指去推卡车——力小、还容易累。真正有劲的是腿和转髋这些大肌肉。所以「传得快」的力学要求就出来了：让大肌肉先发力，从地面借力，把速度沿着身体一节节向上传、逐级放大，最后才交给手。这条一节节向上的传递，就是动力链。

它像一条鞭子：腿和髋是鞭柄（发力、传力），手臂手腕是鞭梢（在最后一刻把攒好的速度「啪」地释放出去）。你甩鞭子时，手只是轻轻一带，力却从柄传到梢、末端炸开——鞭梢的速度远超你的手。传球，就是把你的身体甩成这样一条鞭子。接下来六个环节，就是这条鞭子从柄到梢、依次点火的顺序。

二、第一环 · 抓地：力的源头，从后脚开始

鞭子要甩，得先有个发力的起点。传球的起点不在手，在你的后脚和它踩着的地面。

先把站姿摆对：侧对目标站，不要正对。为什么是侧对？因为力要从地面起、经过一次转体才能放大——你得先把身体「拧」到侧面、留出转过来的空间，正对着目标就没地方转了（这正是 00 课说的「鞭柄要先蓄上劲」）。侧身站好后，把重心压到后脚（投掷手同侧那只脚）上，脚掌稳稳踩实地面。

接着是发力的第一下：后脚蹬地。你往地里一蹬，地面会给你一个大小相等、方向相反的反作用力——这就是整条动力链的力气从哪来：不是你「造」出力，是你从地面借来力。这股从脚底涌上来的力，是后面每一环要传递、放大的那份「本金」。本金越足（蹬得越实），后面能放大出的球速上限就越高。

这一环带走
任务：传得快 → 力学要求：从大地借来一份足够大的「本金」→ 于是动作：侧对目标、重心压后脚、后脚用力蹬地。力从这里起，手臂在这一刻还完全不该动。新手几乎都跳过这一步——脚下没动静，等于鞭子还没握住柄就想甩梢。

三、第二环 · 跨步指向目标：给这股力定个方向

脚下借到了力，但它现在是「向上」的——你得把它拐向目标，还得给身体腾出转起来的空间。这两件事，靠同一个动作完成：跨步（stride）。

做法：前脚（戴手套那侧的脚，也就是非投掷手同侧脚）朝目标方向跨出一步，脚尖指向目标落地。这一步看着简单，却一次干了三件关键的事：

定了方向前脚脚尖指哪，身体接下来就会朝哪转、球最终就往哪去。跨步的方向 = 传球的方向。脚尖歪了，后面转得再好，球也偏。所以「传得准」这一半，其实是脚决定的——又一次印证 02 课那句「靠的是脚」。
把身体拉开、蓄上劲前脚往前一迈，身体被拉长、后侧（髋和躯干）被进一步「拧紧」——鞭柄这时才真正上满了劲，像拉开的弓。没有这一步，你就没有可以「转过来」的行程，链的第三环（转髋）根本无从发动。
做好转的准备前脚落地成了一个支点：身体接下来要绕着这条前腿把髋和躯干「转过去」。落地要稳、别太远也别太近（大约自己一个自然的跨步距离），太远会失去平衡、转不动，太近又蓄不上劲。
这一环带走
任务：传得直、传得准，且要能转 → 力学要求：给地面来的力定个方向、并拉开身体腾出转体行程 → 于是动作：前脚朝目标跨一步、脚尖指向目标、稳稳落地。新手常常原地不跨步就甩手——等于既没定方向、又没给转髋留空间，力只能全靠手臂硬挤，当然又飘又软。

四、第三环 · 髋先转：鞭柄先动，这是全课最关键的一环

现在弓拉满了，该放箭了。放箭的第一下，不是肩、更不是手，而是——髋部先转向目标。请把这句话记牢，它是整条动力链的命门，也是新手最常跳过的一环。

为什么是髋先动？回到鞭子：你甩鞭，是先动鞭柄，力才有东西可传、可放大。髋部（连着腿和核心这些大肌肉）就是鞭柄。它先转，就把脚下借来的那股「本金」接住、并开始向上传。这时候你的肩膀还要故意留在后面——形成一个「髋已经朝前、肩还朝侧」的拧紧状态（髋肩分离）。正是这个「髋领先、肩滞后」的时间差，让力在躯干里被进一步拧紧、放大，就像你把毛巾两头反向一拧，中间会绷出巨大的张力。

顺序为什么必须是「下→上、髋先于肩」？因为这样才能逐级放大又不丢准：大肌肉（腿、髋）力气大但慢，让它们先动、把速度一级级传给灵活的末端；等力传到又快又灵活的手腕时，速度已经被逐级叠加到很高了。顺序一旦颠倒——手或肩抢先发力——大肌肉的力就接不上、传不上去，动力链当场「断掉」。这就是 00 课说的「只甩鞭梢、丢了鞭柄」，也是几乎所有传球毛病的总根源。

这一环带走
任务：把地面的力逐级放大 → 力学要求：大肌肉先动、制造「髋领先肩」的拧紧差 → 于是动作：髋部先于肩膀转向目标，肩暂时留在后面。这是鞭柄先动。把这一环加进去，球速的涨幅比你练一个月手臂都大——因为你终于把最强的大肌肉接进了链条。

五、第四环 · 肩带臂：手臂是被「拖」上来的，不是自己抬的

髋转到位、躯干拧到极致，接下来会发生一件让新手意外的事：你几乎不用「主动」抬手臂，肩和手臂会被躯干「甩」着、「拖」着自己转上来。

这正是鞭子的中段：鞭柄转过来了，鞭身被带动，力一路往梢上跑。具体到身体：髋先转 → 躯干跟着转 → 肩被躯干带过来 → 手臂被肩「拖」着甩上来 → 手落到最后。手臂在这里是「被动跟随」，不是主动发力。你要做的，是让肩和手臂放松地被这股力带走——就像鞭梢是软的，才甩得响。如果你在这一刻还想着「用力抬肘、用胳膊使劲甩」，反而会把柔软的鞭梢绷硬，力传不过去，速度就卡在这儿上不去了。

这也是「省力」的秘密所在：真正出力的是腿和躯干这些大肌肉，手臂只是顺势跟随、在末端做微调。所以用上全身传球，手臂反而更轻松——因为它不再是那个咬着牙硬扛的发动机，而只是一根被甩动的鞭梢。传一整天球都不酸，靠的就是这个。

六、第五、六环 · 鞭打出手 + 随挥：把力干净地放给球，再稳稳收住

力一路传到了鞭梢，只剩最后一下——释放（release）。

第五环 · 鞭打出手：当手臂被带到身体前方的最后一刻，用手腕和手指把球「拨」出去——这就是鞭梢「啪」的那一下，把整条链攒了一路的速度，在这一瞬全部交给球。出手点在身体前方（不是脑袋侧后方就早早撒手），这样速度累积得最满、方向也最正。而手指拨球的方式，正接上 01 课那个接口：食指中指对称地拨过四缝线，给球干净的后旋（backspin）——于是球飞得又快又直。你看，01 课的「握法」到这一刻才真正兑现：前面攒的所有力，全靠这最后一拨，经由那个正确的接口，干净地交给球。

第六环 · 随挥（follow-through）：球出手了，动作却不能戛然而止。手臂要顺势继续向下、向对侧腿的方向送出去，自然地把力「放干净」，而不是急刹车硬停在半空。为什么？两个原因：一是护臂——甩出去的鞭子你猛地一刹，那股冲力会全砸在你的肩肘上，这是手臂受伤的头号来源；顺势收住，力被慢慢卸掉，关节就安全。二是放干净力——不留手、不提前减速，球才拿到了动力链全部的速度。随挥不是多余的收尾，它是「快」和「不伤」这两个任务的共同要求。

1抓地——侧对目标、重心压后脚、后脚蹬地，从地面借来力的「本金」。鞭柄·力源
2跨步指向目标——前脚朝目标跨一步、脚尖指向目标落地，定方向、拉开身体、备好转。定方向
3髋先转——髋部先于肩转向目标，肩留在后面，拧出髋肩分离的张力。鞭柄先动·命门
4肩带臂——躯干和肩被带转，手臂被「拖」着甩上来，放松跟随、别主动抬肘。鞭身传力
5鞭打出手——到身体前方最后一刻，手腕手指拨球出手，给出四缝线后旋。鞭梢·释放
6随挥——手臂顺势向下、向对侧腿送出去，不急刹车，护臂又放干净力。收力·护臂
把六环连成一句话
一次好的上手传球，是一条从脚底点火、依次传到指尖的鞭子：后脚蹬地借力 → 前脚跨步定向 → 髋先转（鞭柄先动）→ 躯干肩把手臂拖上来 → 身前一刻手腕拨球出手 → 顺势随挥收住。大肌肉先发力、手臂最后动、髋永远先于肩——顺序对了，力就逐级放大又不丢准；顺序错了，链就断在半路。

七、新手的头号通病：只抬肘、用手臂甩（push，不是 whip）

现在你有了「正确」这把尺子，就能一眼看穿最常见的错误了。几乎每个没学过的人，第一次传球都会做同一件事：脚不动、髋不转，原地抬起胳膊、抬高手肘，用手臂和肩把球「推」出去。——这就是 push（推），而不是 whip（鞭打）。

用动力链的语言说，它到底错在哪？很清楚：它跳过了前四环里最有力的部分——没抓地（没借到力）、没跨步（没定向、没蓄劲）、更没髋先转（鞭柄根本没动）。于是那条鞭子只剩最末端的一小截在甩，动力链断在了手臂这一环之前。后果全都对得上你一开始的挫败：

  • 球又飘又软。 （只有手臂那点力，速度上不去；手离球太近、幅度又小，稍偏就飘——00 课那张图早就画过：手臂能贡献的那一截又小又固定。）
  • 手臂又酸又累。 （你让最弱的小肌肉去扛整份活儿，还每次都急刹车不随挥，肩肘天天超负荷。）
  • 越使劲越糟。 （你以为是力气不够，于是更用力抬肘——结果是把断掉的那半截鞭子甩得更狠，球更飘、臂更伤。）

解法不是「更用力」，恰恰相反——是把跳过的环节一个个接回去：先蹬地、再跨步、再让髋先转，手臂反而放松下来当鞭梢。你会得到一个反直觉却千真万确的结果：用上全身，球更快，人却更省力。下面这台小玩具，就让你亲手把这些环节一个个接上，看着球速涨、费力度降。

八、动手：把发力环节一个个接上，看球速和费力怎么变

光靠文字记「髋先于肩、手臂最后」还太抽象。下面这台小玩具让你亲手把动力链的环节一个个打开。画面是一个侧视的传球者，从后脚 → 髋 → 肩 → 肘 → 手 → 球分成一节节的鞭子。你可以勾选加入跨步 / 转髋 / 随挥三个环节（手臂始终在用，因为它是末端鞭梢），再切换髋肩顺序对不对。右侧读数会显示这一传的球速（示意）、费力度（越用全身越省力），以及动力链完整度。

发动动力链 · 只用手臂 vs 逐环接上（抓地→跨步→转髋→随挥）
默认只勾了「手臂」——这就是新手「原地抬肘甩」。依次勾上跨步 / 转髋 / 随挥，看侧视鞭子被逐节点亮、出球箭头变长、球速猛涨而费力度反降。转髋是涨得最多的一环。再点髋肩顺序把它调成「肩抢先」，看链怎么当场「断」掉、球速掉下来——顺序错了，环节再全也白搭。球速/费力为示意·量级值，帮你抓「逐环接上→更快更省力、髋要先于肩」这几个因果，不是实测。

多勾几次，那个贯穿全课的事实会自己跳出来：只有手臂时，球速低、费力度高（一个人硬扛）；每接上一个环节——尤其是转髋——球速就往上跳一大截，费力度反而降下去。而只要你把「髋肩顺序」调错成肩抢先，无论环节勾得多全，球速都会掉——因为链断了。这就是「力从地面来、髋要先于肩、手臂最后动」最直白的一张图。记住这个画面：接下来的每一课，都是在帮你把这条蓝色的链接得更顺。

把这一课接回主线

一句话带走
上手传球是一条从脚底点火、甩到指尖的鞭子，按顺序六环：抓地（后脚蹬地借力）→ 跨步指向目标（定方向、蓄劲）→ 髋先转（鞭柄先动，肩留后拧出张力）→ 肩带臂（手臂被拖着甩上来，别主动抬肘）→ 鞭打出手（身前一刻手腕拨球，给四缝线后旋）→ 随挥（顺势收住，放干净力又护臂）。大肌肉先发力、髋永远先于肩、手臂最后动——顺序对了才逐级放大又不丢准。新手的头号毛病是只抬肘用手臂甩（push 而非 whip）：跳过抓地跨步转髋，链断在半路，球又飘又软还伤臂。解法不是更用力，是把环节接回去——用上全身，反而更快更省力。

你现在会把动力链的力甩给球了。可 00 课早就埋了一句话：传球和挥棒，是同一条动力链。挥棒也是蹬地 → 转髋 → 躯干带肩、带手臂——只不过这一次，鞭梢甩的不再是球，而是一根棒，而且多了一道全新的难题：球是飞过来的、动的，你得在对的位置、对的时机让棒头正好和它相遇。传球你只要把力放出去就行；挥棒还要「打得到、打得准、打得结实」。这，就是第 04 课要解决的下一环。

常见误解

  • 误解：传球主要是手臂和肩膀的活儿，抬高手肘用力甩就行。 （澄清：这正是新手头号病「只抬肘用手臂甩」（push）。手臂是末端鞭梢，力气小；真正的发动机是腿和转髋。跳过抓地、跨步、转髋，动力链就断在半路——球又飘又软还伤臂。要 whip，不要 push。）
  • 误解：不跨步、原地站着也能把球传好，跨步是多余的。 （澄清：跨步一步干三件事——定方向（脚尖指哪球去哪）、拉开身体蓄劲、给转髋腾出行程。不跨步就没方向、没劲、也没法转髋，力只能全靠手臂硬挤。）
  • 误解：转髋不重要，把肩和手臂甩快点效果一样。 （澄清：转髋是全课最关键的一环——它是鞭柄先动，把脚下借的力接住并放大。跳过它，最强的大肌肉就没接进链条，球速的最大一截增量直接丢掉。这也是最多人漏掉的一环。）
  • 误解：髋、肩、手臂谁先动无所谓，一起使劲最有力。 （澄清：顺序是命门。必须髋先于肩、肩再带臂、手最后（下→上）。这样力才逐级放大又不丢准；顺序一颠倒（肩或手抢先），大肌肉的力就传不上去，链当场断掉——widget 里把顺序调错，球速立刻掉。）
  • 误解：球一出手动作就该停下，随挥是可有可无的花架子。 （澄清：随挥是「快」和「不伤」的共同要求。不随挥＝球出手前就急刹车，力没放干净（球更软），冲力还全砸在肩肘上（受伤头号来源）。手臂要顺势向下、向对侧腿送出去。）
  • 误解：传不快是因为我手臂力量不够，该多练胳膊。 （澄清：球速的主力是腿和转髋这些大肌肉，不是手臂。很多力气不大的人传得又快又轻松，靠的正是把动力链接顺；只练手臂反而强化「只甩鞭梢」的坏习惯。用上全身，反而更省力。）
下一步
你已经能把动力链的力又快又直地甩给球了。下一步，把同一条链甩给棒——但这次难在球是动的，你得在对的点、对的时机接住它。 → 第 04 课《挥棒与击球点：从「打到」升级到「打准、打结实」》会带你从「同一条动力链（蹬地→转髋→鞭打），只是甩的是棒」出发，再解决那道传球没有的新题：击球点该在身体哪个位置、路径怎么又平又短、时机怎么对上——你会发现挥棒的底层原理你其实已经学过一大半了。
