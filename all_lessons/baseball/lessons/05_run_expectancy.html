第二部分 · 得分怎么发生

垒包与得分期望：四个垒是四档概率（数学心脏）

上一课我们让打者把球结结实实地打了出去——高初速、对的发射角、一记漂亮的飞球。可「打得好」离「得分」还隔着一段路：球落地、跑者上了垒，然后呢？棒球残忍的地方在于，上垒不是得分。一个站上一垒的跑者，必须再被推过二垒、三垒、本垒，踩满四个垒包才算一分。所以问题变成了：站在某一个局面上——垒上有谁、出了几个局——你这半局到本局结束，预计还能拿几分？这一课，我们要把整项运动里最重要的一张表算出来。它是这门课真正的数学心脏。

线性回顾
上一课（04 · 击球的物理）：飞球革命教会我们，把球打到空中（高初速 + 对的发射角）才能把「打中」变成「上垒甚至全垒打」。
留下的问题：除了一棒打过墙的全垒打，绝大多数得分不是一击完成的，而是跑者沿着四个垒一点点推进的结果。那么「上了垒」到底值多少分？同样是一垒有人，0 出局和 2 出局完全是两码事——怎么把这种局面的价值精确地量出来？
本课新增：读完你会掌握得分期望矩阵（Run Expectancy，RE）——把棒球场上的局面抽象成 24 种「垒上有人 × 出局数」状态，每一种都对应一个「到本局结束的期望得分」。有了它，场上发生的任何一件事，都能被翻译成一句精确的话：它让期望得分变了多少。
数据小注
本课表里的所有 RE 数字（如 0 出局满垒 ≈ 2.3、0 出局空垒 ≈ 0.5、2 出局空垒 ≈ 0.1）都是数量级 / 示意值。真实的 RE 矩阵由某个联盟、某段年代的海量历史半局统计平均而来——它随得分环境波动：1990 年代末「类固醇时代」全联盟火力旺，整张表偏高；2010 年代后期投手与三振占上风，整张表偏低；美联（有指定打击）通常又略高于国联。所以请把这些数字当成「典型现代大联盟」的近似锚点，记住形状（谁高谁低、差多少）比记住小数点后两位重要得多。
本课路线
(1) 先看清垒包系统——四个垒是一条单向的推进管道，越靠近本垒，得分越「近」；(2) 引出得分期望矩阵的定义：24 种状态 = 8 种垒上组合 × 3 种出局数，每种一个期望得分；(3) 读这张表，找出它的三条规律（垒越多越值钱、垒越靠前越值钱、出局越少越值钱）；(4) 给出它最重要的用途——RE24：用「事件前后 RE 之差」给每一个事件定价；(5) 把它接回 xG / 每回合期望分那场「期望值革命」；(6) 亲手点选 24 个状态、对比任意两个状态的 RE 差。

一、四个垒：一条单向的推进管道

先把最基础的物理摆清楚。棒球的内场是一个边长约 27.4 米（90 英尺）的正方形，四个角是一垒、二垒、三垒、本垒。一名跑者要得分，必须按逆时针顺序、依次踩过一垒 → 二垒 → 三垒，最后回到本垒。少踩一个、踩错顺序都不算。这是一条单向管道：跑者只能往前，不能后退。

这个简单的几何，带来一个关键直觉：跑者离本垒越近，他得分所需的「后续事件」就越少。一个三垒上的跑者，只要一记不太深的飞球、一次内场滚地、甚至一个暴投，就能轻松回本垒得分；而一个一垒上的跑者，得靠后面接连几次成功的推进才能跑完三个垒。所以——

核心直觉
四个垒，本质上是四档「得分概率」。跑者所在的位置（空垒 / 一垒 / 二垒 / 三垒 / 各种组合），编码的是「他最终踩回本垒的可能性有多大」；而出局数，编码的是「这半局还剩多少次机会去推进他」。把这两件事一组合，每一个局面就有了一个确定的「预计还能得几分」——这就是得分期望。

跑者越多、越靠前，得分越近；出局越少，剩下的机会越多。把这两个旋钮拧到不同位置，棒球场上一共有多少种本质不同的局面？数一数：垒上的有人 / 没人组合是 2³ = 8 种（一垒有没有人 × 二垒有没有人 × 三垒有没有人），出局数有 0、1、2 三种（2 出局后再一个出局就换半局，所以「出局数」只到 2）。8 × 3 = 24。整项运动里所有可能的「跑垒形势」，都落在这 24 个格子里。

二、得分期望矩阵：给 24 种局面各贴一个期望得分

现在做一件朴素到近乎笨、却威力惊人的事。回到几十年的历史比赛记录，把每一个半局拆成一连串「状态」。比如某个半局：开局是「空垒 0 出局」，打者一垒安打 → 变成「一垒有人 0 出局」，下一棒三振 → 「一垒有人 1 出局」……一直到第三个出局、半局结束。

对其中每一种状态，我们都问同一个问题：历史上，所有「曾经处在这个状态」的半局，从这个状态起算、到本局结束，平均一共又得了几分？把这个平均值贴在那个格子上，就得到下面这张得分期望矩阵（行 = 垒上组合，列 = 出局数；值为示意）：

把这张表看明白，你就拿到了整门课后半程的钥匙。先确认那三个被点过名的锚点：空垒 0 出局 ≈ 0.50（一个全新的半局，从无人出局、无人在垒开始，平均能得半分多一点——这也正是「场均得分 / 9 局」的雏形）；满垒 0 出局 ≈ 2.29（三个跑者全部就位、一个人还没出局，这是进攻能遇到的最肥的局面，预计能得两分多）；空垒 2 出局 ≈ 0.10（两人已经出局、垒上空空，这半局基本要黄了，预计连零点一分都不到）。

三、读出三条规律：表里写着棒球的全部物理

这张表不是 24 个孤立的数字，它有清晰的结构。盯着它看几眼，三条规律会自己浮出来——而这三条，恰好就是第一节那条「单向推进管道」直觉的量化版：

  • 规律一：垒上人越多，越值钱（往下走，数变大）。同一列里，从空垒到满垒，RE 单调上升。多一个跑者，就多一个潜在的得分点。
  • 规律二：垒越靠近本垒，越值钱（同样几个人，位置越靠前越高）。对比「三垒 0 出局 ≈ 1.30」和「一垒 0 出局 ≈ 0.86」：同样是一个跑者、同样 0 出局，跑者在三垒比在一垒多值近半分——因为他离本垒近，得分所需的后续事件少。这就是「四个垒是四档概率」的直接读数。
  • 规律三：出局越少，越值钱（往右走，数暴跌）。这是最陡的一条。看任意一行：满垒从 0 出局的 2.29，掉到 1 出局的 1.54，再掉到 2 出局的 0.75——每多一个出局，期望得分几乎腰斩。空垒那一行更夸张：0.50 → 0.27 → 0.10。

第三条规律，请你格外用力地记住。它在说一件反直觉的事：「出局」是这张表里最贵的东西。多一个跑者、把跑者往前挪一个垒，期望得分往上动零点几分；而失去一个出局，期望得分往下掉的幅度，常常比这大得多。出局是有限的（一会儿你会在下一课看到，每队每场总共只有 27 个），用一个出局去换垒上的进展，到底划不划算？——这正是 06 课要算的账，这张表就是算账的工具。

四、最重要的用途：RE24，给每一个事件定价

到这里，这张表真正的用处才登场。它不只是「描述局面有多好」，它能给场上发生的任何一件事定价。方法朴素到只有一步减法：

RE24 一句话定义
任何一个事件（安打、保送、出局、盗垒、牺牲触击……）的价值，等于它带来的得分期望变化：
RE24 = (事件后状态的 RE) − (事件前状态的 RE) + (这次事件中实际跑回本垒的分数)
它把模糊的「这球打得好不好」，变成一句精确的话：「期望得分 +0.3」还是「−0.2」。（名字里的「24」就是指这 24 种状态。）

用几个例子把它跑活：

  • 一垒安打把跑者从一垒送上三垒（0 出局）。状态从「一垒 0 出局」(0.86) 变成「一·三垒 0 出局」(1.65)，没人得分。RE24 = 1.65 − 0.86 + 0 = +0.79。一次漂亮的安打，凭空给球队增加了近 0.8 分的期望。
  • 一个普通的出局，垒上无人（0 出局 → 1 出局）。状态从「空垒 0 出局」(0.50) 变成「空垒 1 出局」(0.27)。RE24 = 0.27 − 0.50 + 0 = −0.23。一个不起眼的出局，悄悄拿走了 0.23 分。出局之所以贵，这就是数字证据。
  • 三垒跑者靠一记牺牲飞球回本垒得分（1 出局 → 2 出局）。状态从「三垒 1 出局」(0.95) 变成「空垒 2 出局」(0.10)，但实打实得了 1 分。RE24 = 0.10 − 0.95 + 1 = +0.15。注意：哪怕真的拿到了 1 分，它对剩下半局的期望其实是净拿走了不少（局面从 0.95 跌到 0.10）——这种「换分但掏空局面」的张力，正是下一课的主线。

这就是 RE 矩阵被称作「数学心脏」的原因：它把棒球这串离散事件，全部翻译成了同一种货币——期望得分的增减。安打值多少、保送值多少、一次出局亏多少、一次盗垒成功赚多少 / 失败亏多少，全都能用同一把尺子量出来、加起来。模糊的「手感」「气势」「关键时刻」，第一次有了可以累加、可以对账的底层信号。

五、又一场「期望值革命」：三大球类殊途同归

如果你读过姊妹的足球课和篮球课，这一刻应该有强烈的既视感。RE 矩阵做的事，和它们做的是同一个思想动作：

棒球 · 得分期望矩阵 / RE24给每一种跑垒局面贴一个「到本局结束的期望得分」，再用「前后之差」给每个事件定价。本课主角。
足球 · xG 期望进球给每一脚射门按位置 / 角度 / 防守人数定一个 0–1 的进球概率。从「射了几脚」升级到「这些机会本该进几个」。见 足球第 12 课。
篮球 · 每回合期望分给每一次出手按位置定价（底角三分、篮下值钱，长两分最亏），把胜负还原成「回合 × 每回合期望分」。见 篮球课。

三项几乎不相干的运动，在不同年代、由不同的人，各自独立地把同一行高中数学——期望值（概率 × 价值，再求和）——认真地用了一遍，每一次都掀翻了「数原始数字 + 老法师拍脑袋」的旧秩序。想看清这个工具本身（什么是期望、为什么求和能抹平噪声），可以回到《数学的逻辑》第 13 课（概率）；想理解「价值 vs 价格」的对照，可以看《市场的逻辑》——RE 之于一次进攻，正如「价值」之于一只股票：它是更稳的底层信号，单场的实际得分（价格）会围着它上下抖，长期向它回归。棒球之所以是这场革命里被改写得最彻底的，是因为它天生就是一连串离散、可统计的对决——一座现成的概率实验室。

多点几下，那个最反直觉的事实就会自己跳出来：把同一行从左往右点（只是多了一两个出局），RE 几乎腰斩；而把同一列从上往下点（多了跑者、跑者更靠前），RE 才慢慢往上爬。这意味着——保护出局，远比急着推进跑者重要。一次贸然的跑垒、一记「为了送跑者而主动送出去的出局」，在这张表上常常是净亏损。这正是下一课要正面算的那笔账。

一句话带走
上垒不是得分；得分是跑者沿四个垒推进的概率游戏。把局面抽象成 24 种「垒上组合 × 出局数」状态，每种都有一个「到本局结束的期望得分」（RE 矩阵）——这是整门课的数学心脏。它写着三条规律：垒越多越值钱、垒越靠前越值钱、出局越少越值钱（且这一条最陡）。靠 RE24（事件前后 RE 之差 + 实得分），任何事件都能被翻译成「期望得分 +0.3 还是 −0.2」。这和足球 xG、篮球每回合期望分，是同一场把「数个数」升级成「按期望价值定价」的革命。
下一步
这张表里最陡、最贵的维度是出局。而出局又是有限的：每队每场总共只有 27 个，用完就结束——它是棒球场上最硬的货币。那么那些「主动献出一个出局换一个垒」的经典战术，到底划不划算？ → 第 06 课《出局的经济学：27 个出局是预算，触击与盗垒划算吗》会用这张 RE 矩阵一格一格地算账，得出一个让无数老派教练不舒服的结论：大多数牺牲触击、大多数贸然盗垒，都是亏本买卖——因为它们拿最稀缺的出局，去换一个本就不那么值钱的垒，期望得分常常不升反降。这是全课最反直觉的一课。
