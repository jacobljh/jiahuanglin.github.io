第一部分 · 规则如何制造游戏

低比分与运气：为什么冰球像足球一样低分、高方差、门将主导

上一课我们把冰面封闭起来——无出界让比赛连续不停，无摩擦让它开到极速，越位线又约束了进攻怎么进区。结果就和足球惊人地相似：射门满天飞，进球却得来不易，一场终了常常只是 3:2、2:1。可「进球稀少」这件事本身，藏着一个比任何战术都冷酷的后果。同样是联盟第一对阵垫底队，冰球里爆冷频频；季后赛七场里一支热手门将能把一轮系列赛整个偷走——这不是谁不努力，是数学。这一课，我们把「进球稀少」翻译成它真正的含义：高方差，以及被它顶到台前的那个人——门将。

本课路线
(1) 先量级对齐——冰球每队每场约进 3 球，和足球一个量级，远低于篮球；(2) 借足球那套泊松直觉（交叉回扣 《足球的逻辑》第 03 课）：均值低 → 单场相对方差大 → 爆冷常见、常规赛强队也连败、季后赛短系列「什么都可能发生」；(3) 冰球的独特放大器——门将（goalie）是低分运动里被顶到极限的单点变量，一个热手门将能偷走一轮系列赛（为第 09 课埋线）；(4) 一个更底层的伏笔——进球既稀少又随机，用进球数评价球队/球员就会非常嘈杂，这正是冰球数据革命 Corsi / xG 的动机（为第 10–12 课埋线，和足球 xG 同一逻辑）；(5) 亲手拨一个泊松比分模拟器，看「略强的一方」到底能赢下多大比例。
数据小注
本课所有数字都是数量级 / 示意，不同赛季、不同规则年代会变。常被引用的「每队每场约 3 球」来自近些年 NHL 的场均进球（两队合计常在 5.5～6.5 之间徘徊，折合每队约 2.7～3.2 球，随赛季与规则调整波动）。把进球当成泊松过程是一个「足够好」的近似模型，不是物理定律——真实比赛里进球并非严格独立（领先方会收缩、拖到最后一分钟拉下门将 6 打 5 搏命、罚球会短暂改变人数）。但即便如此，泊松对真实进球分布的拟合好得出奇，足以让我们把「低分」翻译成「高方差」。凡涉及具体球员/门将的说法，我们只讲机制，不编造精确数字。

一、先把量级对齐：冰球也是「个位数进球」的运动

先把三项运动的进球「产量」摆在一起看，冰球的位置立刻清楚。

篮球：几十上百分（高分）一场每队七八十到一百多分，得分事件密如雨点。样本极大，运气来不及作妖——强队几乎必赢。
足球：每队一两球（低分）九十分钟每队约 1.3～1.5 球（场均合计约 2.7，示意）。进球是镶嵌在漫长「无事发生」里的稀疏几点。
冰球：每队约 3 球（低分）六十分钟每队约 3 球（场均合计约 6，示意）。射门可以三四十次，但绝大多数被扑、被封、被打偏——真正入网的仍是个位数。

关键在于：冰球虽然射门比足球密得多（一场每队三四十次抽射稀松平常），但进球依旧停在个位数。上一课已经讲过为什么——极速的封闭冰面让空间转瞬即逝，射门容易、好射门难，而门将又高大、护具占满球门。于是冰球和足球坐在同一格里：它是一项个位数进球的低分运动。而「低分」这三个字，在数学上等价于一个你在看台上天天见到、却很少细想的词——高方差。

二、泊松直觉：为什么均值越低，运气分到的权重越大

把进球想成一种「稀疏的随机事件」：六十分钟里绝大多数时刻什么都没进，进球是零星几个点。当一种事件满足「每一小段时间发生概率都小、不同时段近似独立、平均发生率大致稳定」，它的计数就近似服从泊松分布 (Poisson)。足球进球是这样，冰球进球也是这样。

泊松分布只由一个参数决定——平均值 λ（一支球队一场「期望进几个球」）。它有一条正是本课要害的性质：

泊松的核心性质（和足球第 03 课同一条）
对泊松分布，方差恰好等于均值：方差 = λ。于是相对波动幅度（标准差 ÷ 均值）是 √λ / λ = 1/√λ——λ 越小，相对波动越大。冰球一队期望进 λ ≈ 3 球，标准差约 √3 ≈ 1.7 球；也就是说，「期望进 3 球」的球队，单场进 1 球、2 球、5 球都稀松平常。均值不高，就把结果的相对随机性顶得很高。

这就是为什么冰球和足球一样「不讲道理」，而篮球不会。篮球场均得分 λ 很大，相对波动 1/√λ 很小——多打几十个回合，强弱被平均出来，强队几乎必赢。冰球每队才进个位数，1/√λ 大得多。样本太小，运气来不及被平均掉。注意：冰球的 λ ≈ 3 比足球的 λ ≈ 1.4 略高，所以冰球的相对方差比足球略小一点点——但和篮球比，两者都属于「高方差」这一类，爆冷都是常态。

把这条性质翻译成你在冰场边真实见到的东西，有三个直接后果：

后果一：常规赛里，强队也会莫名其妙连败。一支球队某一场的期望进球是 λ，可具体到这一场，泊松让它有真实的概率被「冻」在 1 球甚至 0 球（λ = 3 时整场颗粒无收的概率约 e^(−3) ≈ 5%——一个赛季 82 场，这种「被零封」会发生好几次）。连着两三场手感不佳，强队就吞下一波连败——这几乎不说明实力变了，只说明样本小、运气还没被磨平。

后果二：射门数会骗人，进球数才算数——但进球数又太随机。这是冰球和足球共享的最反直觉的一点。射门次数衡量「你制造了多少机会」，可进球是机会乘以一个很低、又很随机的转化率（远射的入网概率可能只有 xG ≈ 0.03，近门的高质量机会也不过 xG ≈ 0.2 上下，示意）。抽了四十脚劣质远射，期望进球也许还不到 2 个；对手两次贴身好机会就够了。数量骗人，质量才真——可即便质量占优，稀少 + 随机仍可能让你这一场吃瘪。（怎么给一次机会的质量精确定价？那个数字就叫 xG，是本轨道后半程的主角，思想直接借自足球。）

后果三：季后赛七场短系列，比 82 场常规赛随机得多。这是「样本量」最赤裸的体现。82 场常规赛是 82 次抽样的平均，运气被磨平，积分榜顶端几乎总是真强队。可季后赛是七局四胜——样本量只有个位数。在这么小的样本里，1/√n 那股巨大的相对波动毫无遮拦地暴露出来。这就是为什么「季后赛什么都可能发生」几乎成了定律，也是为什么常规赛碾压的头号种子，第一轮就被下克上，年年都有。

交叉链接 · 同一个思想，几个地方
「均值低 → 相对方差大 → 小样本里运气压过实力」不是冰球独有的把戏，是概率论的通则。最纯粹的数学版本见 《数学的逻辑》· 概率（期望、方差、为什么稀有事件服从泊松）。在市场里它同样搅动人心——一个基金经理某年跑赢大盘，多少是本事、多少是 n=1 的噪声，见 《市场的逻辑》关于运气 vs 实力、方差与样本量的讨论。而姊妹运动给出了绝妙对照：《足球的逻辑》第 03 课讲的是同一条泊松（足球 λ 更低、方差更大）；《棒球的逻辑》则反着来——用一个赛季 162 场的超长赛程主动给运气降噪。场次越多，1/√n 越小，实力越显形；冰球和足球偏偏进球少、（在季后赛里）场次也少，于是把运气的权重保留到了最大。

三、冰球独有的放大器：门将，被顶到极限的单点变量

低分运动有一个足球也有、但冰球把它推到极致的角色——守门的那个人。想想这条因果链：进球本就稀少（每队约 3 个），而这稀少的几个球，全都要穿过同一个人。于是这个人的当晚状态，几乎单枪匹马地决定了比赛的随机性走向哪一边。

把它和泊松接起来看：如果某位门将今晚「热」了——扑救成功率 save% 从平常的 91% 涨到 94%，听起来只差 3 个百分点，可对方抽四十脚的话，这意味着从被打进约 3.6 球压到约 2.4 球。在一项每队只进 3 球的运动里，凭空省下一球多，往往就是胜负本身。门将不是在「减少失球数」，他是在直接改写这场比赛的 λ。

为什么冰球的门将比足球的更「单点」
两项运动都低分、都有门将，但冰球把这个变量顶得更高，原因有二：其一，射门量大得多——足球门将一场可能只面对几脚有威胁的射门，冰球门将要面对三四十次抽射，样本更大、他的当晚 save% 更能持续地压制或释放对方的进球；其二，护具与球门尺寸让扑救本身成了一门可以「爆种」的技艺，一个门将完全可能在一轮系列赛里连续打出远超常态的表现。结果就是冰球界那句老话：一个热手门将能偷走一轮系列赛。这条线，我们在第 09 课《门将：最后一块空间的管理者》会专门展开——他既是防守体系的最后一环，也是低分 + 高方差这台机器里最大的那颗骰子。

这里先记住方向就够了：低分让运气的权重变大，而门将是那份运气最集中的落点。一支球队可以在阵容、战术上样样占优，却在七场系列赛里输给对面门将的一波神勇——这不是玄学，是「稀少 + 单点」两件事叠在一起的必然。

四、真正的伏笔：既然进球又少又随机，就别用它当尺子

把前面三节连起来，会逼出一个不安的结论。如果一支球队的实力，被泊松那股 1/√λ 的噪声、又被门将这个单点变量反复搅动，那么用「进了几球 / 赢了几场」去评价一支球队或一名球员，就是在用一把抖得厉害的尺子量东西。一名前锋这个月进了 8 球，多少是他真的变强、多少只是运气把几脚该被扑出的球送进了网？光看进球，你分不清。

这正是冰球数据革命的起点，也是它和足球 xG 共享的同一个念头：既然「结果」（进球）太稀少太随机，就往上游找一个更频繁、更稳定的信号。冰球找到了两个——

Corsi：数「射门尝试」而非进球把一支球队所有的射门尝试（射正 + 打偏 + 被封堵）都算进来，衡量它「掌控了多少进攻」。进球一场只有个位数，射门尝试却有几十次——样本大了十倍，噪声就被压下去。Corsi 高的球队，长期看几乎总是更强。
xG：给每一次机会按「质量」定价不只数机会的数量，还按射门位置、角度、类型给每次机会算一个入网概率，加起来就是「这场你理应进几球」。它把「射了很多烂球」和「制造了几次真机会」区分开——思想和足球 xG 一模一样。

你不必现在就懂它们怎么算——只要记住动机：进球太稀少，就得找更底层的信号。这几课（第 10–12 课）会把 Corsi 与 xG 一层层讲透，并让你看到它和足球 xG、篮球每回合期望分、棒球 wOBA 其实是同一个思想的几次独立发现：给每个事件按「它对得分的期望贡献」定价，绕开那个又稀少又骗人的最终比分。

五、亲手拨一拨：泊松比分模拟器

光说不练没有说服力。下面这个模拟器让你扮演两支球队的「期望进球 λ」。把主队设得比客队略高（比如 3.0 对 2.6，主队就是那支「略强」的一方），然后连续模拟几百上千场——每一场，程序都按各自的 λ 用泊松抽一个比分。看读数：略强的一方到底能赢下多大比例？平局（在冰球里会进加时/点球决胜）和爆冷又占多少？

你会看到一件值得记住的事：哪怕主队的 λ 明显更高（理应更强），它的常规时间胜率也远到不了 100%——平局会吃掉一块（这些在真实比赛里被加时和点球大战拆开，本身就是纯运气的加时赛），爆冷（客队赢）也总有可观份额。这就是「低分 → 高方差」的全部含义：实力是真的，但单场里运气分到的权重大得惊人。再做个对照：保持主队按比例更强，把两队 λ 一起调高（想象一项高分运动，比如 8.0 对 6.5），再看胜率——主队会赢得「干脆」得多，平局被大幅挤掉。这反过来证明了：是冰球的低分，而非别的什么，制造了它独有的不确定性。

多玩几把你会发现：把主队 λ 设到 3.0、客队 2.6，主队常规时间胜率大约落在四成半到五成，打平约六分之一，客队仍能赢下接近三成半——「略强」换来的优势，远没有直觉以为的那么大（那些打平的比赛，还要靠加时和点球大战来分胜负，而那几乎是纯运气）。而当你保持主队按比例领先，把进球数整体推高（比如 8.0 对 6.5），胜率就明显更干脆、平局被大幅挤掉：进的球越多，单场样本越「大」，运气越被平均掉，强弱越分明。注意一个反直觉的细节：如果你只是把两队 λ 同时加上 0.4（绝对差不变），优势其实几乎不动——真正决定强弱能否显形的，是「相对」领先多少、以及进球这个样本有多大。冰球的低分，正是它把运气权重顶满的根本原因。

六、把这一课接回主线

到这里，链条又往前扣了一环：封闭冰面（第 01 课）让比赛连续又极速、射门满天飞却好机会难求，于是进球停在个位数；而个位数进球在数学上等价于高方差——爆冷常见、强队连败、季后赛短系列格外随机（本课）。冰球还比足球多了一个放大器：门将，那份运气最集中的单点落点。也正因为进球又少又随机，我们才不得不往上游去找 Corsi / xG 这样更稳的信号——那是本轨道后半程的伏笔。

一句话带走
冰球的低比分不是缺陷，是它高方差魅力的数学源头。进球近似泊松，而泊松方差 = 均值意味着均值越低、相对波动越大（1/√λ）。所以强队会连败、射门多的会输、季后赛尽是冷门——这是「低分 = 高方差」的必然，不是谁不努力。冰球还把这份运气集中到一个人身上：门将，一个热手门将足以偷走一轮系列赛。而正因进球又少又随机，用它当尺子太抖，我们才需要 Corsi 和 xG。
下一步
低分源于进球难，进球难又源于空间难得——可空间为什么这么难得？答案是冰球把足球的空间难题乘上了速度。 → 第 03 课《空间与速度：速度如何让空间更稀缺，传球与滑行制造空当》会告诉你：冰面上人和球都开到极速，防守回位更快、你做决策的窗口更短，于是空间比足球更稀缺、也更转瞬即逝。制造空间的杠杆和足球一样（无球滑行、传球调动、拉开宽度与纵深），但一切都必须更快完成——这就是冰球真正的核心难题。
