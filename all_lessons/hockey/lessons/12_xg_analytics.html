第四部分 · 数据革命

期望进球 xG 与现代分析：冰球 xG 直接借自足球

上一课，Corsi / Fenwick 用「射门尝试差」给你补上了一台仪表盘——不再盯着稀少又随机的进球，而是数那个多得多、稳得多的先行指标：谁在射、谁在挨射，用它代理控球与领地。可 Corsi 有一个你一定已经想到的硬伤：它把每一次射门尝试都当成一样的——蓝线外一脚 40 英尺、被三个人挡着的碰运气远射，和 slot 里一次门户大开的补射，在 Corsi 眼里都只是「+1」。可你我都知道，这两脚的价值天差地别。数次数，不看质量。那怎么把「这一脚到底有多可能进」也算进来？冰球的答案不是自己发明的——它直接借来了足球的 xG。这一课，是整个「球类的逻辑」系列数据革命同源最直白的证据。

本课路线
(1) 先点破 Corsi 的硬伤——只数次数、不看质量（回扣 11）；(2) 给出 期望进球 xG 的定义：给每一脚射门，按位置、角度、射门类型、是不是补射 / 反击，算一个 0～1 的进球概率；(3) 摊开本课最重的一拍——冰球的 xG 与足球的 xG 几乎是同一个模型，并把整条系列脊柱点全（足球 xG ↔ 篮球四要素 ↔ 棒球 wOBA ↔ 橄榄球 EPA ↔ 冰球 xG，同一个思想：给每个事件按对得分的期望贡献定价）；(4) 三样应用——球员的 xG 贡献、门将的 GSAx（扑救超越期望，回扣 09）、球队的 xG 差与高危射区地图；(5) 诚实地说 xG 是模型、有噪声、冰球样本比足球更少更需谨慎；(6) 亲手拨一个「拖射门点看 xG」的部件——把射门位置在冰面上拖来拖去，看门前高、外围低，再切补射 / 反击看它怎么被抬起来。
数据小注
本课所有 xG 数字都是数量级 / 示意值，用来建立直觉，不是任何一家模型的真实输出。真实的冰球 xG 由 MoneyPuck、Evolving-Hockey、Natural Stat Trick 等各家，用不同特征（射门到球门的距离、角度、射门类型 wrist / slap / tip、是不是 rebound、是不是 rush、前一事件、赛况……）在历史数据上训练出来；同一脚射门在不同模型里会差零点零几。GSAx（Goals Saved Above Expected）也是各家口径略有出入的派生量。本课 widget 用的是一个刻意简化、但方向正确的几何模型（只看距离和角度，再叠加补射 / 反击的乘子），门前高、外围低的结论是真的，具体小数请别当成权威统计。绝不编造精确数据。

一、Corsi 的硬伤：数次数，不看质量

先把上一课的成果和它的裂缝一起摆正了看。Corsi 干了一件了不起的事：它绕开了「进球太稀少、单场几乎测不准球队实力」这个死结，改去数一个多得多、也稳得多的东西——射门尝试（射正 + 射偏 + 被封堵）。一场球两队合计几十上百次射门尝试，样本大，噪声小，于是 Corsi 差（你射了多少 − 对方射了多少）成了控球与领地的好代理：谁常年把比赛压在对方半场，谁的 Corsi 就正。

但请回想 Corsi 是怎么数的——它把每一次射门尝试都记成同样的一票。于是荒唐的事就来了：

一脚 40 英尺的蓝线远射从进攻区最外围、极小的角度，硬轰一脚，中间还隔着两三名封堵的防守者和门将清晰的视线。它真正进球的概率可能只有 xG ≈ 0.02～0.03——五十脚里进不了一个。可在 Corsi 账本上，它是干干净净的 +1。
slot 里一次门户大开的补射门将刚扑出第一脚、还趴在冰面上，你在球门正前方五英尺抢到弹回来的球，对着半个空门补第二枪。它的进球概率可能高达 xG ≈ 0.4 以上。可在 Corsi 账本上，它也只是 +1。

两脚射门，进球期望差了十几倍，Corsi 却给它们记了一模一样的分。这就是 Corsi 的硬伤：它衡量的是「你制造了多少次射门尝试」（数量），却完全无视「每一次尝试到底有多好」（质量）。一支专打劣质远射的球队，Corsi 可能很漂亮，进球却寥寥；一支射得少、但每一脚都从门前高危区出手的球队，Corsi 平平，得分却凶狠。光看 Corsi，你会把这两支队看反。

问题因此逼得清清楚楚：我们需要一个数字，能给每一脚射门单独定价——它到底有多可能变成进球？这个数字，冰球没有另起炉灶，而是从一项和它血缘最近的运动那里，整套借了过来。

二、期望进球 xG：给每一脚射门标一个进球概率

这个数字叫期望进球（expected goals，xG）。它的定义朴素到近乎显然：

xG 的定义
给每一脚射门，根据它出手时的处境，算一个 0～1 之间的数，表示「像这样的一脚射门，历史上平均有多大比例最终进了球」。这个数，就是这一脚的 xG。xG = 0.15 的意思是：把成千上万脚「和它长得差不多」的射门放在一起，大约 15% 进了。把一支球队一场里每一脚射门的 xG 加起来，就得到「按机会质量算，这场你应该进几个球」——这叫球队的 xG（有时写作 xGF，expected goals for）。

那模型看哪些「处境」来给一脚射门定价？冰球 xG 主要吃这么几样特征——它们联手决定了那颗球有多可能钻进网窝：

位置（离球门多远）最要命的一维。球门正前方那片扇形的 slot 高危区，射门距离近、可打的网窝角度大，xG 高得多；越往外围、越靠蓝线，距离一拉开，xG 就断崖式往下掉。门前五英尺和蓝线四十英尺，是两个世界。
角度（正面还是死角）正对球门，你能瞄准的网窝最宽；被逼到球门侧面、贴着底线的死角，能进球的缝隙窄成一条线，门将也好封。同样的距离，正面的 xG 远高于极限角度。
射门类型（怎么打的）手腕射（wrist）快而准、大力抽射（slap）力量大但慢、折射 / 挑射（tip / deflection）改变球路让门将来不及——不同打法的历史进球率不同，模型把它当一个特征。
是不是补射 / 反击（门将来得及吗）这一维最能体现「门将状态」。补射（rebound）：门将刚扑出、位置全乱、门户大开，同样位置的 xG 陡增；反击（rush）：防守还没落位、门将来不及调整站位（回扣第 06 课的 odd-man rush），xG 也显著更高。这正是第 07 课「门前混乱抬高 xG」量化后的样子。

把这几维喂进一个在海量历史射门上训练的模型，它就能对任何一脚新射门吐出一个 xG。于是 Corsi 的硬伤被补上了：那脚 40 英尺蓝线远射标 0.02，那次门前补射标 0.45——射门尝试终于不再是同样的一票，每一票都按它的质量称了重。

三、系列脊柱最直白的证据：冰球的 xG，就是足球的 xG

现在到了这一课、乃至整个「球类的逻辑」系列最想让你看见的一拍。上面这套 xG 的思路，你如果读过足球，会有强烈的既视感——因为它压根就是从足球搬过来的。

同一个模型，两项运动
冰球的 xG 与足球的 xG，几乎是同一个模型。足球给每一脚射门按「离球门的距离、角度、用脚还是头、有没有过掉门将、防守人数」算一个进球概率（详见姊妹课 《足球的逻辑》第 11 课 · xG）；冰球给每一脚射门按「离球门的距离、角度、射门类型、是不是补射 / 反击」算一个进球概率。换掉几个特征名字，两个模型就重合了。这绝不是巧合：足球和冰球是一对双子——都是连续、低分、争空间的运动（回扣本轨道 00、02、03 课）。同样是「进球稀少 + 一脚射门的价值天差地别」，同样的问题，逼出了同样的解法。冰球分析圈当年就是直接看着足球的 xG把它搭起来的。

而这对双子，只是一条更长的脊柱上的两节。整个系列的数据革命，其实是同一个思想在五项运动里的五次独立发现——把它一次点全：

1足球 · xG（期望进球）——给每一脚射门按位置 / 角度 / 部位 / 防守，算一个进球概率，衡量机会质量而非射门数量。《足球的逻辑》11。
2篮球 · 真实命中率 / 四要素——别数「投进几个」，要按每次出手的期望得分定价（三分×命中率 vs 中距离），四要素（投篮效率 / 失误 / 篮板 / 罚球）拆解每回合价值。《篮球的逻辑》11。
3棒球 · wOBA——别把所有上垒都当一样，给每一种结果（一垒打 / 全垒打 / 保送……）按它平均贡献多少得分加权，算一个更真的进攻价值。《棒球的逻辑》11。
4橄榄球 · EPA（每次进攻的期望分变化）——每一档进攻，按它把「这波持球的期望得分」推高或压低了多少来定价，而不是只看推进了几码。《美式橄榄球的逻辑》11。
5冰球 · xG（本课）——给每一脚射门按位置 / 角度 / 类型 / 补射反击，算一个进球概率，直接借自足球。

五项运动，五套术语，可你若把它们并排一读，会发现它们说的是同一句话：

系列脊柱 · 一句话
给每一个事件（一脚射门、一次出手、一次击球、一档进攻），按它对「最终得分」的期望贡献定价——而不是只数它发生了几次。这就是所有现代球类分析的共同起点。它和《市场的逻辑》里「别看价格看价值、给每一份现金流按期望贴现定价」是同构的想法（《市场的逻辑》），背后的数学，则是《数学的逻辑》里的期望：把一件随机的事，用「所有可能结果 × 各自概率」加权成一个可比较的数（《数学的逻辑》）。冰球 xG 之所以能直接从足球搬来，正因为它们共享的从来不是「足球 / 冰球」的表皮，而是这条「按期望贡献定价」的骨。

四、xG 拿来干什么：球员、门将、球队三个尺度

有了给每一脚射门定价的 xG，它立刻在三个尺度上都变得有用——从一个人，到一个门将，到一整支球队。

尺度一 · 球员的 xG 贡献。把一名前锋一个赛季所有射门的 xG 加起来，就得到「按机会质量，他应该进多少球」。拿他实际进的球去比：实际 > 期望，说明他要么手感火烫、要么终结能力确实超群（也可能只是这个样本里运气好）；实际 < 期望，说明他浪费了不少好机会（或者运气差）。更妙的是，xG 还能公平地夸奖那些不进球却一直在制造好机会的球员——他们持续产出高 xG 的射门和传球（xA，expected assists，期望助攻），哪怕进球暂时没来，xG 也证明了他的价值。这和足球用 xG 给前锋「解冤」是一模一样的用法。

尺度二 · 门将的 GSAx（扑救超越期望）。这是 xG 最漂亮的一个衍生用途，也正好接上第 09 课门将那一课留的尾巴。回想那里的难题：门将的扑救率（save%）会骗人——一个门将 save% 高，可能是他真神，也可能只是他身前防守好、他面对的尽是些外围软射。怎么把「他到底比一个普通门将强多少」剥出来？答案就是 xG：

GSAx：门将版的「实际 vs 期望」
把一个门将面对的每一脚射门的 xG 加起来，就得到「一个联盟平均水平的门将，面对这一堆射门，预计会丢几个球」（因为 xG 之和 = 期望进球数 = 期望丢球数）。再看这个门将实际丢了几个。GSAx = 期望丢球 − 实际丢球：正得越多，说明他扑掉了越多「按理该进」的球，越是真神。举个示意例子：他这段时间面对的射门 xG 之和是 28.0（平均门将预计丢 28 球），他实际只丢了 22 球——GSAx = 28 − 22 = +6，他替球队凭空省下了 6 个球。这就把「他身前防守好不好」这个干扰项先扣掉了：无论别人喂给他的是软射还是高危射，都已经折进 xG 里。GSAx 因此比裸的 save% 公平得多，是当今评价门将的主流指标之一（回扣第 09 课：门将真正的价值 = 扑救超越期望的那部分）。

尺度三 · 球队的 xG 差与高危射区地图。把 Corsi 差换成 xG 差（xGF − xGA，你的期望进球 − 对方的期望进球），你就有了一台比 Corsi 更聪明的仪表盘：它不光看你把比赛压在哪半场，还看你在那半场造出的机会值不值钱。一支球队可以 Corsi 领先（射门尝试多）却 xG 落后（尽是劣质远射）——这台仪表盘一眼就把它看穿。更直观的是高危射区地图（shot / heat map）：把一支球队所有射门按 xG 画在冰面上，红的热区若集中在 slot，说明它在最值钱的地方出手；若散在外围一圈，Corsi 再好看也是虚的。从 Corsi（数量）到 xG（质量），是同一台仪表盘的一次精度升级。

五、诚实地说：xG 是模型，不是真理

把 xG 捧到这里，得赶紧泼一盆冷水——它是个极有用的工具，但它是模型，不是从天而降的真理，用它得带着几分警惕。

xG 是概率，不是判决一脚 xG = 0.15 的射门没进，模型没「错」——它本来就说「大约六次里进一次」。xG 描述的是一类射门的长期平均，对单独一脚永远是概率，不是「这球该进 / 不该进」的裁决。拿它去审判某一个具体瞬间，是误用。
它看不见的东西很多大多数公开 xG 模型看不到门前有没有遮挡、传球来得多刁钻、防守者具体站哪、门将的重心偏向——而这些恰恰能大幅改写真实概率。所以同一个 xG 值，背后可能是很不一样的处境。模型只吃它被喂的那几维特征。
冰球样本比足球更少、更需谨慎这是冰球 xG 尤其要小心的一点。冰球虽然射门尝试比足球多，但进球更稀少、比赛更混乱（碰撞、遮挡、折射、门将扑救的随机性），可训练的「干净」样本相对更薄，各家模型的分歧也更大。xG 在冰球里需要更大的样本才收敛，单场、甚至十几场的 xG 差，噪声都还很重——别拿一场的 xG 去下重注。

所以正确的姿势是：把 xG 当成一副校正过的、但仍会抖动的眼镜——它让你看清了 Corsi 看不见的「质量」这一维，这是巨大的进步；但它不替你做判断，样本不够时更要收着信。它是这门课教你的最后一件、也是最精密的一件看球工具：不再数进球（太随机），不再只数射门（不看质量），而是给每一脚射门按它值多少期望进球来称重。

六、亲手拨一拨：拖射门点看 xG

下面这个部件是一片冰球进攻区，球门在顶端。中间有一个橙色的射门点，你可以把它拖到冰面上任何地方。右边的读数会实时用一个简化的几何 xG 模型（只看你离球门的距离和角度）告诉你：从这里射门，示意的 xG 是多少。你还能拨两个开关——补射（门将刚扑出、门户大开）和反击（防守没落位、门将没站好，回扣第 06 课）——看同一个位置的 xG 怎么被门前 / 转换的混乱抬起来。

把射门点拖进球门正前方的 slot，你会看到 xG 蹿到很高（示意 0.19 上下，越贴近球门越高）；往外拖，xG 断崖式往下掉——拖到约 40 英尺的外围正面一脚，xG 只剩 0.02～0.03，再退到蓝线边缘更是低到几乎为零；而把它拖到球门侧面的死角，哪怕离得不远，xG 也因为角度变窄而明显下滑。这就是「位置 + 角度」两维的全部直觉：门前高、外围低、正面高、死角低。然后打开补射或反击——射门点一步没动，xG 却整体被乘高了一截：这正是第 07 课那团「门前混乱」被 xG 量化后的样子。射门尝试终于不再是同样的一票——你手里这个数，就是每一脚射门的价签。

拖射门点看 xG · 门前高、外围低，补射 / 反击再抬一截
拖动橙色的射门点在进攻区里移动，右边实时显示这一脚的示意 xG（只看距离 + 角度的简化几何模型）。把它拖进球门正前方的 slot 看 xG 蹿高，拖到蓝线 / 死角看它跌到很低。再拨 补射（门户大开）和 反击（门将没站好）两个开关，看同一位置的 xG 被抬起来。所有数字为示意值，非任何模型真实输出。点「复位」回到 slot 中路、无补射无反击。

没有 canvas 也不影响理解：想象进攻区顶端一个球门。站在球门正前方几英尺、正对网窝射门——门将来不及、可打的网窝又宽，xG 高（示意 ≈ 0.2）。退到四十英尺开外的外围硬轰一脚，距离一拉开、角度也小，xG 跌到 ≈ 0.02～0.03，再退到蓝线边缘几乎归零。若换成一次补射（门将扑出后趴着）或一次反击（门将还没站好），同一个位置的进球概率会被抬高一截。位置、角度、混乱——这就是 xG 的三味主料。

这个几何模型当然是刻意简化的（真实 xG 还看射门类型、传球来向、防守人数、门将站位……，且是在海量历史数据上学出来的，不是我这几行三角函数），但它的方向和结论都是真的，也和足球的 xG 完全同构：给每一脚射门按它的进球概率定价，位置和角度是主料，门前 / 转换的混乱是能显著改写它的调味。你现在手里拨的，就是那把补上了 Corsi 硬伤的、给射门称重的秤。

七、常见误解

  • 误解：xG 高的射门没进，就说明 xG 「算错了」。 （澄清：xG 是概率不是判决——xG = 0.2 本来就说「五次里大约进一次」，这一次没进完全正常。它描述一类射门的长期平均，对单独一脚永远是概率。）
  • 误解：冰球 xG 是冰球分析师自己发明的高深玩意。 （澄清：它直接借自足球——两者都是连续、低分、争空间的双子，同一个「给射门按进球概率定价」的模型换几个特征名就通用。这正是整个系列数据革命同源的最直白证据。）
  • 误解：Corsi 好的球队一定强，xG 是多此一举。 （澄清：Corsi 只数射门数量，一支专打劣质远射的队 Corsi 可以很漂亮却 xG 落后。xG 补上了「质量」这一维——它是 Corsi 的升级，不是重复。）
  • 误解：门将 save% 高就是好门将，GSAx 没必要。 （澄清：裸 save% 会被「他身前防守好不好、他面对的射门难不难」污染。GSAx 先用 xG 把射门难度扣掉，再看他比平均门将多扑了几个——这才是他自己的贡献，回扣第 09 课。）
  • 误解：单场 xG 差就能定谁踢得好。 （澄清：冰球比赛混乱、进球稀少，xG 需要较大样本才收敛，单场噪声很重——别拿一场的 xG 差下重注。这是冰球 xG 比足球更要谨慎的地方。）
一句话带走
Corsi 只数射门次数、不看质量，于是冰球借来了足球的 xG：给每一脚射门，按位置、角度、射门类型、是不是补射 / 反击，算一个 0～1 的进球概率——门前 slot 高、外围死角低。它补上了 Corsi 的硬伤，还派生出门将的 GSAx（扑救超越期望，把身前防守的干扰扣掉）和球队的 xG 差。冰球的 xG 与足球的 xG 几乎是同一个模型——这是整个系列数据革命同源最直白的证据：足球 xG ↔ 篮球四要素 ↔ 棒球 wOBA ↔ 橄榄球 EPA ↔ 冰球 xG，同一个思想——给每个事件按对得分的期望贡献定价。诚实地说：xG 是模型、有噪声、冰球样本更少更需谨慎。
下一步
我们已经有了给射门与球员定价的整套工具（Corsi、xG、GSAx）。可一支球队从来不是把几个高 xG 的明星堆在一起就行——冰球换人不停、每个人上冰时间有限（回扣第 05 课换人），所以它比任何单项运动都更吃深度与角色分工。那怎么把一群球员拼成一台真正能跑起来的机器？ → 第 13 课《建队与阵容：四条前锋线 + 三对后卫的深度》会告诉你：冰球阵容不是堆明星，而是配一台有深度的机器——四条前锋线各有分工（得分线、两能线、防守线、体力线），三对后卫、外加门将；因为换人不停、上冰时间有限，深度和角色分工比哪项运动都关键。
