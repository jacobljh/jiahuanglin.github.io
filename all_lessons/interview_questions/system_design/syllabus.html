System Design Interview syllabus

A PDF-backed case layer for the existing 19-lesson system-design track. The public URL stays here; this page links the 44 case-study drills and shows how the Alex Xu books and archive are covered without duplicating them.

Recommendation
Keep System Design - Interview Prep as the series. Do not create a parallel top-level track. The current 19 lessons are the foundation layer; this syllabus adds the case-study layer underneath the same URL.
