第一部分 · 规则如何制造游戏

导览：跑了上百公里，只进两三个球，到底在争什么？

90 分钟里，场上 22 个人来回奔跑，双方合计跑动往往超过 100 公里（英超一支球队单场约 110 公里，示意）。可终场哨响，比分常常只是 1–1、2–1。一边是上百公里的疯狂奔跑，一边是稀得可怜的进球——如果进球这么少，这 90 分钟里这么多人到底在争什么？这门课，就是为了把这个问题一步步挣明白。答案只有两个字：空间（以及与它共生的时间）。

如何读这门课
这门课不是 16 篇互不相干的「足球小知识」，而是一条推理链。我们从一个谁都看过、却从没细想的反差出发——跑得这么多、进得这么少——每一课都由上一课没答完的问题逼出来。读到最后，你不是背下了一堆战术名词，而是亲手把「足球为什么长这个样子」一步步推了出来。请按 00 → 15 的顺序读：跳着读会断链，因为每一课都站在前一课的肩膀上，回答它留下的那个缺口。
数据小注
本课与全程用到的数字，多为示意 / 数量级，不同赛季、不同联赛、不同统计口径都会变，凡是会变的我们都会标出来。例如：英超近年场均进球约 2.6 ~ 2.9 球（折合每队每场 1.3 ~ 1.5 球，随赛季波动）；一支球队单场跑动约 100 ~ 115 公里。这些数字的量级是稳的（进球以「个」计、跑动以「百公里」计），细节请别当成精确统计。绝不编造小数点后的「权威数据」。
本课路线
(1) 用「跑上百公里只进两三球」这个钩子，把全程的中心引擎立起来；(2) 把它说清楚——足球的一切都来自三重规则约束（禁手 + 大场地 + 人数相等），它们联手把「空间与时间」变成最稀缺的资源；(3) 给出贯穿「球类的逻辑」整个系列的三元对照（足球=连续 / 篮球=回合 / 棒球=离散），并交叉链接两门姊妹课；(4) 画出全 16 课的完整推理链与两条线索（空间的创造线 / 利用线）；(5) 亲手玩一个迷你半场——拖动一名无球队员，看防线被他牵着走、一块没人看管的空当被生生拉开。

一、钩子：跑了上百公里，只进两三个球

先把那个反差摆正了看。一场顶级联赛，全场22 名球员持续奔跑、变向、冲刺，两队加起来跑过的距离常常超过一个马拉松（100 公里出头，示意）。可这 90 分钟生产出来的「成品」——进球——往往只有两三个。换算一下：球员每跑几十公里，才换来一粒球。这在任何别的「产出」眼里，都低得离谱。

更怪的是：进球这么稀少，比赛却一点都不闷，全世界几十亿人为它疯狂。这说明一件事——这 90 分钟里被疯狂争夺的，根本不是「进球」这个结果本身（它太稀有了），而是别的东西。那些奔跑、传递、卡位、回追，绝大多数都没直接产生进球，可它们显然在争某种资源。

那个资源，就是空间：球场上一块暂时没人看管、能让你安全持球或接球的地方。以及与它一体两面的时间：你在被对手扑到之前，能从容处理球的那零点几秒。进球之所以稀有，正是因为对方时时刻刻都在剥夺你的空间与时间；而你这 90 分钟做的每一件事，本质上都是为了在某个瞬间，抢回一块空间、偷出一点时间。看懂足球，就是看懂空间如何被创造、转移、利用。

可空间为什么会变得这么金贵？为什么不像篮球那样动不动就得分？答案不在球员身上，而在规则里。

二、立起中心引擎：三重约束，把空间逼成最稀缺的资源

现在把整门课要挣来的那句话完整写出来——后面的课程会一步步把它变成你自己推出来的结论：

全程中心引擎
足球的一切，都来自三重规则约束的合谋——禁手（不能用手，只能用脚和身体，处理球天生低效）＋巨大的场地（约 105 × 68 米，比篮球场大十几倍）＋人数相等（双方各 11 人，没人天生多）。这三条联手把空间与时间变成全场最稀缺的资源，于是进球极其罕见。正因如此，所有的战术、跑位、阵型、数据与审美，都是为了同一件事：在对的时间、对的地点，制造出一个「多出来的人」（局部以多打少）或一块「没人看管的空间」。看球 ＝ 看空间如何被创造、转移、利用。

逐条拆一下这三重约束为什么必然导向「低效、低分、空间为王」：

禁手 → 低效手是人类最精密的工具，脚不是。强迫你只能用脚控球、传球、射门，每一次触球的失误率和难度都陡然升高。把球安全地从 A 送到 B，本身就难——这是低分的第一个根源。
大场地 → 空间难守11 个人要看管一块巨大的草坪，人均要负责的面积大得吓人。哪儿都顾不全，处处是缝隙；但反过来，缝隙也处处被人盯着——空间于是成了一场永不停歇的拉锯。
人数相等 → 必须「制造」优势双方各 11 人，谁也不天生多一个人。所以你想要的「以多打少」从不会凭空出现，只能在局部、在某个瞬间被制造出来——靠跑动、靠传球、靠把对手骗动。这是全程所有战术的总目标。

三条加在一起，结论无可回避：把球安全送到对方门前太难（禁手），门前那点空间又被严防死守（大场地 + 人数相等），于是进球必然稀有。而既然结果如此稀有，比赛的真正内容就只能是对过程性资源——空间与时间——的连续争夺。这台引擎，会驱动你接下来读的每一课。

三、三元对照：每种球类，都是规则人为制造的一种稀缺

这台引擎其实不止属于足球。它是整个「球类的逻辑」系列共享的同一个思想：每一种球类运动，都是一套规则人为制造出来的稀缺与冲突；看懂一项运动，就是看懂它的规则把「什么」变成了最稀缺的资源，于是所有战术都只是对那个稀缺的理性回应。三项运动恰好坐在同一设计空间的三个角上：

足球：把「空间 / 时间」变稀缺（连续流动）规则（禁手 + 大场地 + 人数相等）让进球极少（场均约 2.7，示意）。比赛是一条不停的河流，没有天然的「回合」边界，资源（空间）连续地被创造和摧毁。这门课讲的就是它。
篮球：把「每一次进攻回合」变稀缺（回合制）规则（24 秒进攻时限 + 三秒区 + 三分线）把比赛切成一段段回合，让每回合期望分变成货币；一切是「每回合榨出最高期望分 / 把对手压到最低」。详见姊妹课 《篮球的逻辑》。
棒球：把「出局数」变成预算（离散对决）规则（三振 / 四坏 / 每队每场只有 27 个出局、且没有时钟）把比赛拆成一连串离散的、一对一的投打对决；整项运动是「用概率管理出局与垒包」。详见姊妹课 《棒球的逻辑》。

把它们并排看，足球的特点立刻清晰：它是三者里最连续、最难切片的一个。篮球能数「这回合得几分」，棒球能数「这次对决出不出局」，足球却几乎数不清「这一脚到底值多少」——因为它是流动的、无界的。正是这份连续与稀缺，让足球既最难用数字描述，又在后来被一个叫 xG 的想法（第 11 课）悄悄改写。这三门课的数据革命，其实是同一个思想的三次独立发现，我们会反复互相回扣。

顺带一提：这台「把一门学科讲成一条推理线」的引擎，也驱动着它的几门远房姊妹课——《烹饪的逻辑》（风味是被操纵的一场构建）、《作曲的逻辑》（音乐操纵你听到的张力与期待）、《电影的逻辑》（电影是一台受控幻觉机器），以及讲价值与价格、运气与方差的 《市场的逻辑》。它们都在说同一件事：别背事实，去追那条把事实串起来的线。

四、两条线索：空间的创造线与利用线

既然全场都在争空间，那么「关于空间，你只能做两件事」：把它创造出来，再把它利用掉。这门课的主线，正好分成这两条彼此咬合的线索——

创造线（把空间生出来）无球跑动 → 宽度与纵深 → 用传球移动对手 → 用阵型分配空间 → 在攻防转换的瞬间偷取失衡的空间。每一步，都在草坪上撕出一块原本不存在的空当。
利用线（把空间换成进球）定位球（把混乱暂时变成可设计的离散战术）→ 一对一过人（在局部制造 2 打 1）→ 临门一脚 → 最后用 xG 给「这块空间到底值多少」定价。每一步，都在把已经造出的空间，兑现成得分的机会。

这门课最深的一拍，就是这两条线其实是同一件事的两半：足球高手做的一切，都是在「制造一个多余的人 / 一块空地」，再赶在它消失前把它用掉。现在，先用一个最小的例子，亲手感受「空间是可以被人凭空创造出来的」。

五、动手：一次无球跑动，如何凭空拉开一块空当

下面这个迷你半场上，5 个蓝色防守者守着一片区域，还有 1 个橙色的无球进攻队员（他脚下没有球）——你可以拖动他。规则极简，却抓住了足球最核心的直觉：离他最近的一两名防守者，会被他「吸引」过去一点（防守者天然要盯住威胁）。于是，你每把这名无球队员拖到一个新位置，整条防线就被他牵着变形，原来站满人的地方，会被生生拉出一块没人看管的空当。

右上角的读数，是当前被拉开的最大无人区（示意值）：我们把半场切成网格，找出离所有防守者都最远的那一块，用它到最近防守者的距离估一个「空当半径」，再换算成一个示意面积。你会看到：当橙色队员把防守者从某一侧勾走时，另一侧的空当会肉眼可见地变大。这就是「无球跑动」的全部秘密——他自己不碰球，却用自己的位置，替队友造出了一块可以接球的空间。

空间是可以被人移动创造出来的 · 拖动无球队员，看防线被牵走
拖动橙色的无球队员（脚下没球的那个）。离他最近的 1–2 名蓝色防守者会被吸引、朝他平移一点；其余防守者不动。右上读数实时算出被拉开的最大无人区面积（示意值）——把他拖到一侧，注意看另一侧的空当怎么变大。点「复位」把所有人放回初始站位。

没有 canvas 也不影响理解：想象一片半场上站着 5 名防守者；你让一名不带球的队友斜插到右路，最近的两名防守者本能地跟过去补防，于是左路原本被盯住的那块草坪，突然没人管了——你的另一名队友就能从容地在那里接球。无球跑动没碰球，却造出了空间。这，就是足球。

这个玩具简化得近乎粗暴（真实防守是 11 人的整体协同，吸引规则也复杂得多），但它的方向是真的：一次聪明的无球移动，能在不碰球的情况下，把全队的空间格局改写。整门课接下来要做的，就是把这个直觉一层层做厚——从一个人的跑动，到 11 个人的阵型，再到给「空间值多少」算出一个数字。

六、这条线索如何串起整门课

每一课都不是新话题，而是上一课没答完的问题逼出来的。下面把全 16 课列成一条「问题 → 回答 → 新问题」的链：

1导览（本课）。——立引擎：三重约束（禁手 + 大场地 + 人数相等）把空间逼成最稀缺资源，进球因此稀有，一切为了制造「多余的人 / 空地」。可这三条约束里，最根本的是哪一条？为什么「不能用手」就足以让整项运动低分、让空间登上主角？
2禁手与大场地。——脚比手低效，处理球难、失误多；再摊到一块巨大的草坪上，空间天然守不过来。这是低分的根。对照：若能用手会变成橄榄球 / 篮球那样的高分运动。就算我能把球带到对方门前，为什么还是这么难得手？因为有一条看不见的线。
3越位。——一条看不见的线，禁止进攻者「躲在最后一名防守者身后等球」，把「在哪里等球」也变成了稀缺。它重塑了整个进攻空间。这么多约束叠在一起，进球到底稀有到什么程度？稀有会带来什么后果？
4进球为何稀少。——低分运动的数学后果：进球近似泊松分布，运气占比大、爆冷常见、一两球定胜负（回扣 math 概率 / markets 方差）。既然进球这么稀少，这 90 分钟到底在「生产」什么？答：空间与机会。那空间怎么造？
5创造空间。——无球跑动、宽度、纵深：为什么「无球的人」往往比「持球的人」更重要——是他们在撕开防线。空间造出来了，怎么把球安全地送到那块空地去？
6传球与控球。——把球当作「移动对手」的工具：tiki-taka 的本质不是炫技，是用连续传球逼对手不停移动、暴露空当。一两个人的配合有限。11 个人要怎么摆，才能系统地占据空间？
7阵型。——4-4-2 / 4-3-3 / 3-5-2 不是站位图，是空间的初始分配与权衡（攻守人数守恒：这边多一个，那边必少一个）。阵型是静态的。可比赛最危险的时刻是动态的——球权易手那一瞬，会发生什么？
8攻防转换。——失去 / 赢得球权的瞬间，空间最失衡：反击的科学，由守转攻那几秒钟，是制造空当的黄金窗口。连续的比赛里，有没有「能预先排练好」的时刻？有——死球。
9定位球。——角球 / 任意球 / 点球：把连续的混乱暂时冻结成可设计的离散战术（点球转化率约 0.76，示意）。这一拍恰好呼应棒球的离散对决。定式之外，靠什么在局部硬生生破局？个人能力。
10个人技术与一对一。——过人的本质：在局部制造以多打少（1 个人晃过 1 个人 = 瞬间 2 打 1）。进攻一直在造空间。那防守这边一直在干嘛？
11防守。——把空间压扁：盯人 vs 区域、防线整体移动、造越位陷阱、门将作为「最后一块空间的管理者」。我们一直在说「好机会」。可怎么客观衡量一次机会到底有多好？
12xG 期望进球。——用一个数字给「机会质量」定价（射门位置 / 角度 / 身体部位 / 防守人数 → 进球概率）。足球版的「价值 vs 比分」。核心回扣篮球每回合期望分、棒球 wOBA、markets 价值 vs 价格。有了给空间定价的语言，现代教练怎么系统地把空间赢回来？
13高位逼抢与传控。——战术史 = 对空间理解的不断升级：克鲁伊夫全攻全守 → 瓜迪奥拉传控 → 克洛普反抢（「逼抢是最好的组织者」）。要执行这些体系，需要什么样的球员？
14位置的进化。——位置是「空间任务」而非站位：清道夫 → 自由人 → 伪九号 → 内收边后卫 → 出击型门将。这些球员从哪来、怎么被买卖？球队其实是一个组织。
15联赛、转会与金钱。——球队作为组织：为什么钱能买强、却买不了确定性（回扣 markets 反身性：身价 ↔ 表现 ↔ 身价）。如果钱、数据、战术都收敛了，我们到底在看什么？
16收官：我们到底在看什么。——回望整条链（空间稀缺 → 创造 / 利用 → 定价 → 系统）；低分 + 爆冷 + 不确定性正是它的魅力（高方差是特性不是 bug）；个人即兴与系统设计的张力。三元对照收束（连续）→ 邀你去读篮球（回合）与棒球（离散）。

全 16 课分成五个部分。它们正好对应那两条线索——前半程学会创造空间，后半程学会利用并给空间定价，最后退一步看整个系统与文化：

七、常见误解

  • 误解：足球主要是关于「谁脚下技术好、谁会盘带」。 （澄清：脚下技术只是利用线的一环。决定一支球队的，更多是无球时的跑位与空间组织——场上 90% 的时间你脚下没有球。足球是一项关于「不带球的 10 个人」的运动。）
  • 误解：控球率高就该赢。 （澄清：控球本身不是目的，是移动对手、撕开空当的手段。控了 70% 却造不出空间、打不出机会，照样输——第 05、12 课会细说，反击型球队常用很低的控球率赢球。）
  • 误解：进球少 = 比赛沉闷、运气成分太大、不值得认真看。 （澄清：低分确实意味着高方差、爆冷常见（第 03 课），但这恰恰是足球的魅力来源，不是 bug。被争夺的「空间」每分每秒都在发生，只是它不像进球那样显眼——这门课就是教你看见它。）
  • 误解：这门课每篇讲一个战术名词，可以挑着看。 （澄清：它是一条链。跳读会断在「上一步没答完的问题」上——请按 00 → 15 的顺序走，每一课都在回答前一课留下的那个缺口。）
一句话带走
跑上百公里只进两三球，争的从来不是「进球」本身（它太稀有了），而是空间与时间。足球的一切都来自三重规则约束——禁手 + 大场地 + 人数相等——它们把空间逼成最稀缺的资源；于是所有战术，都是为了在对的时间地点，制造一个多余的人或一块空地。看球 ＝ 看空间如何被创造、转移、利用。这就是后面 15 课的发动机。
下一步
三重约束里，最根本的是第一条。先把它弄明白——为什么「不能用手 + 场地巨大」必然导致低效率、低比分，从而让空间登上整项运动的主角席。 → 第 01 课《禁手与大场地：低效率如何让「空间」成为主角》会带你看清：手是人类最精密的工具、脚不是，强迫用脚就让每次触球都变难、变险；再把这份低效摊到一块巨大的草坪上，进球就必然稀有。作为对照，它还会告诉你：如果允许用手，足球会立刻变成橄榄球或篮球那样的高分运动——稀缺，从来都是规则设计出来的。
