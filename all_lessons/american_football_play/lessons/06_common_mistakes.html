第三部分 · 玩起来（真的开始上场）

常见错误的自查与纠正：每个坏习惯都是动力链断在某一环

到这里，00–05 你已经把全部动作学完了：怎么握球、怎么站、怎么掷螺旋、怎么接、怎么护、怎么跑位。可你一走上场地真练，多半会发现一件泄气的事——脑子里都懂，手上一做就走样：球又开始翻跟头、扔得又软又近、接球总弹开、跑起来一被人碰球就掉。别急，也别以为是自己没天赋。这一课不教任何新动作，它给你一样别的东西：一张诊断表。新手最常犯的六个错误，每一个我们都把它还原成同一句话——你的动力链（或两个人的配合）断在了某一环；然后告诉你那一环是哪一课讲过的、怎么当场发现自己正在犯、以及一条对症的纠正练习或口诀。学会自查，你就不再需要教练站旁边盯着——你自己就是自己的教练。

本课路线
(1) 先说清这一课的用法——它是一台诊断仪，把「球怎么又不对了」这种模糊的挫败，翻译成「动力链的第几环断了」这种能动手修的具体问题；(2) 逐一拆解六个最常见的坏习惯，每个都按同一个四步模板讲透——症状（你会看到什么现象）→ 根因（断在哪一环、回指哪一课）→ 自查信号（怎么当场抓到自己）→ 纠正练习 / 口诀；(3) 给你一张随身的总自查清单和一句能背走的口诀；(4) 亲手勾选你身上有的坏习惯，眼看着一次完整回合（传→接→带）的成功率怎样被一环环拖垮、最拖后腿的是哪一环；最后交棒第 07 课——毛病会挑了、会修了，就该真的上场打第一场了。
怎么用这一课
请把它当工具书，不必一次背完。练球时哪儿不对了，回来对照六条症状，找到你那一条，照着「自查信号」当场验证、照着「纠正练习」修。关键心法只有一句：所有这些五花八门的毛病，根子都是同一个——力从地面 / 跨步传上来、由手指拨给球这条链（以及和同伴的时空配合）在某一环断了。你不是有六个孤立的问题，你是有一条链，它可能在六个地方之一松脱。看懂这一点，纠错就从「碰运气乱试」变成了「哪断修哪」。
数据小注
本课 widget 里的「成功率」「各环扣分」等数字，是示意 / 量级，只为让你直觉地感到「每多一个坏习惯，就掐断一环、成功率就往下掉一截」，并看出哪一环最要命。它不是某人某次训练的实测统计。请记住那个「勾得越多、掉得越狠，且总有一环最拖后腿」的趋势，别记具体百分比。

一、满握没螺旋：球翻着跟头飞

第一个，也是最扎眼的一个。你终于敢用点力扔了，球却翻着跟头出去——头尾乱翻、像一块木头砸出去，飞不远、还老是砸不准。这不是力气问题，是球根本没转起来。

症状球出手后翻跟头 / 打滚，不是那种拧着劲、绕长轴自转的漂亮螺旋。飞行发飘、距离短、砸到人身上会「啪」地弹开而不是稳稳钻进手里。
根因：断在末端（第 01 课）你多半是满把攥死了球、掌心闷在球上，食指压根没搭在鞋带上、也没留出让球滚出去的空隙。于是出手最后一刻，食指拨不出那个「搓转」——动力链前面攒的力都到了，却卡在最后一环，旋转给不出来。链的末端（手指→球）断了。
自查信号盯着自己的手：出手瞬间，食指有没有从鞋带上「刮 / 拨」下去的感觉？如果只觉得整只手掌把球「推」了出去、指尖没有那个最后一勾，就是它。慢放看球：翻滚 = 没拨；打旋 = 拨到了。
纠正练习 / 口诀先检查握法：手指压鞋带、掌心和球留一道缝（回 01 课）。再做慢动作空拨——不追求远近，只极慢地把出手最后 20% 做出来，专注「食指最后拨下鞋带」。口诀：「掌心留缝、食指最后一勾」。近距离对墙轻抛，只看球转不转，转了再加力。

二、不跨步、只用胳膊甩：球软而近

这是全系列从第 00 课就点名的头号通病，几乎每个新手都犯。你站着不动，光靠胳膊抡一下把球甩出去。结果：球软趴趴的、飞不远，而且你自己小臂特别酸。

症状球又软又近，没速度、没穿透力；扔完发现累的是胳膊（小臂发酸），而不是腿和核心。别人接你的球毫不费劲，因为它「飘」过来。
根因：断在起点（第 03 课）你跳过了地面和腿——没跨步、没蹬地、没转髋转肩，直接从胳膊这个末端鞭梢发力。动力链的发动机（腿+躯干）根本没启动，力从起点就断了，胳膊那点小力气当然又累又弱。
自查信号扔完看你的脚：如果后脚没有往前跨、重心没有从后脚移到前脚，人还杵在原地，就是只用了胳膊。再摸一下：累的是小臂还是大腿？累小臂＝断在起点。
纠正练习 / 口诀先迈步、再扔——强制自己「不跨步不许出手」。做夸张跨步练：把跨步做到有点滑稽的大，感受重心整个压过去。对墙分解慢练：跨步→转髋→转肩→最后才甩臂，一环一环来。口诀：「先迈步、后甩臂，力从脚下起」。

三、用胸接球：球「啪」地弹飞

球传过来，你一紧张，往往拿身体去挡——胸口一迎，想用身体把球「兜」住。结果球「啪」地一声弹开，掉在地上。这是接球环节最常见的第一个坑。

症状球总是砸在胸口、然后弹飞，怎么都兜不住；越是重要的球越想用身体挡，也就越容易弹掉。手常常是接球时才慌忙去捞，已经晚了。
根因：不能卸力（第 04 课）胸口是又硬又不会动的地方，球撞上去没有任何缓冲，动能原样反弹——就像球砸墙。接球的本质是「卸掉来球的力」，而只有能往后收、能让开的手才卸得掉。用胸接，等于用最不会卸力的部位去硬挡。
自查信号回想球弹开的那一下：球先碰到的是你的手，还是胸 / 身体？如果是身体先碰到，就是它。还有一个信号：接球时你的手有没有「随球往后收」的动作？完全没收、硬邦邦地等球，多半就在用身体挡。
纠正练习 / 口诀只用手接鸡蛋——想象来球是个鸡蛋，你得用手轻轻接住并顺势往后收一点，把力泄掉，绝不能硬碰。近距离两人轻抛，规定「球只能碰手，碰到身体算失败」。口诀：「用手不用胸，接住往回收」（回 04 课的手型 + 卸力）。

四、不看球入手、提前抬头：漏接

这个错误特别隐蔽，因为你自己常常没意识到。球快到手了，你却提前抬头去看跑哪、看有没有人追——眼睛一离开球，球就从指缝里漏掉了。

症状莫名其妙的漏接：球明明到位了、手也伸对了，却没接住。事后你甚至说不清球是怎么没进手的——因为在球入手那一刻，你的眼睛已经不在球上了。
根因：眼睛离球（第 04 课）接球是手眼配合，手要往哪收、什么时候合拢，全靠眼睛一路盯着球进手（look it in）来校准。你提前抬头，等于让手在「盲接」——配合的眼睛这一环断了，手再对也白搭。急着往下一步跑，恰恰毁了这一步。
自查信号漏接后诚实地问自己：「我真的看见球进到手里了吗？」如果答案是「没有，我当时在看别处 / 已经想着跑了」，就是它。另一个信号：你是不是球还没到就开始转身 / 抬头。
纠正练习 / 口诀出声给自己下令：「看进手，再跑！」——把「看球入手」和「起跑」的顺序强行分开、并说出来。练习时刻意盯着球一直到它被收进腋下夹好（tuck），再抬头。口诀：「盯到收进腋下，才准抬头」（回 04 课的看球入手 + 收球）。

五、护球太松 / 单手拎球：一碰就掉

你好不容易接住了球，开始带球跑，却单手把球拎在身侧、晃来晃去。触身 / 腰旗里虽然不许猛撞，但防守一伸手一拨，或者你自己一变向，球就掉了。

症状带球时球拿得松、在身侧晃；被人一碰、或自己一变向就掉球。跑直线还行，一加速、一转身就出事——最可惜的丢球往往就在这儿。
根因：五点包球断了（第 02 课）你没做到high-and-tight（球高高夹在腋下、贴紧身体），也没用五点把球锁死（手指扣尖、手掌托、前臂贴、球身贴肋、球尖藏进腋下）。球在护牢这一环上是松的——跑得再快，链的这一环一断，球就守不住。
自查信号低头看你带球的样子：球是贴紧身体夹在腋下，还是单手拎在离身体老远的地方晃？让同伴轻轻拨一下你的球——一拨就动 / 就掉，就是没夹牢。变向前你有没有下意识把球夹得更紧？没有的话，就是它。
纠正练习 / 口诀把 high-and-tight 变成肌肉记忆：接住球立刻夹进腋下、五点锁死。练「变向前先夹紧」——每次要切、要转身之前，先把球往身上再压一下。找人轻拨球练抗掉，练到怎么拨都拨不掉。口诀：「high-and-tight，变向前先夹」（回 02 课的护球）。

六、跑位不看传球手、切得拖沓：永远对不上

最后一个，是两个人之间的错。你自己动作都对，可一跟人配合传接，就永远对不上——不是球到了人没到，就是人到了球没到。这不是谁的手不行，是你俩的时空没对上。

症状总是差那么一点：球扔到了你本该在的位置，你却还没跑到；或者你切出去了、回头找球，球却还没来。跑位拖泥带水、切得不干脆，节奏和传球手永远差半拍。
根因：时空 / 提前量断了（第 05 课）传接是两个人的配合：传球手得把球送到你将要到的位置（lead the receiver），你得在约定的步数 / 时机干脆地切、并回头找球。只要路线、时机、提前量任一环两人没对齐，配合就断——单项技术再好也串不成一次成功传接。
自查信号每次没接到，判断是「球到人没到」还是「人到球没到」：前者＝你切慢了 / 跑拖了；后者＝提前量或时机没约好。还有：你切的时候是干脆一蹬就变向，还是拖着圆弧慢慢拐？拖沓＝时机必然错。
纠正练习 / 口诀切要干脆——在固定步数后一蹬急停 / 急转，别拖圆弧（回 05 课）。跑中就回头找球，别切完才慢慢找。和传球手先约好节奏：数几步切、往哪切、他在你切的哪一刻出手。口诀：「切要脆、跑中找球、和他对好拍」。

七、动手：勾上你的坏习惯，看它们怎样拖垮一回合

六个毛病单看都懂了，但真正的杀伤力在于——它们会叠加。一次完整的回合是「传 → 接 → 带」一条链走下来，只要其中任何一环断了，这一回合就崩在那儿。下面这台小玩具让你亲手体会这件事。左边有几个开关，对应上面几个坏习惯：满握没螺旋、不跨步只用胳膊、用胸接球、不看球入手、护球太松。你勾上哪些，就等于亲手掐断对应的那一环；画面上那条「传→接→带」的链会把断掉的环节标红并注明是哪一课，右边的读数会显示这一回合的综合成功率怎样往下掉、以及最拖后腿的是哪一环。按「▶ 来一回合」还能看球到底崩在哪一步。

你会看到一件和第 00 课呼应的事：全不勾时，一条干净的链走下来，成功率很高；每多勾一个坏习惯，就掐断一环、成功率就往下掉一截；勾得越多、掉得越狠。而且总有一环最拖后腿——那就是你现在最该先修的地方。这正是这一课想交给你的自查方式：别笼统地怪「我不行」，去找是哪一环断了，然后哪断修哪。

诊断台 · 勾上你的坏习惯，看一回合怎样崩
勾选你身上有的坏习惯（对应上面的前五条——第六条「跑位对不上」是两个人的配合，不在这条单人的「传→接→带」链里），每勾一个就掐断动力链 / 配合的一环：看右边本回合综合成功率怎么掉、最弱的一环断在哪（并回指课号）、以及螺旋 / 接球等子项状态。按▶ 来一回合看球在「传→接→带」里崩在哪一步。全不勾＝干净回合、成功率最高；勾越多掉越狠，且总有一环最拖后腿——那就是先修它。成功率、扣分为示意/量级值，帮你抓「每个坏习惯 = 断一环」这个直觉。

八、把这一课接回主线

一句话带走
新手场上那些五花八门的毛病，根子是同一个：力从地面 / 跨步传上来、由手指拨给球这条动力链，以及你和同伴的时空配合，在某一环断了。满握没螺旋＝断在末端出手（01）；只用胳膊＝断在发力起点（03）；用胸接、不看球＝断在接球的卸力与看球（04）；护球松＝断在护牢（02）；跑位对不上＝断在两人的提前量与时机（05）。所以纠错不是碰运气乱试，而是——先把「球又不对了」翻译成「哪一环断了」，再哪断修哪。学会自查，你就成了自己的教练。

会挑毛病、会自己修，这门课的「练」就闭环了。可练来练去，终究是为了真的玩一场。你现在缺的，不再是某个动作，而是把它们放进一场真实对局所需要的那点东西：怎么热身不受伤、两个人第一组对练怎么练、触身 / 腰旗最基本的规则和安全边界、场上最起码的礼仪，以及——练完这八课之后，接下来该按什么顺序继续变强。这正是最后一课要给你的。

常见误解

  • 误解：我总犯这些错，是因为没天赋 / 力气不够，练也白练。 （澄清：这六个几乎全是习惯问题，不是天赋问题——满握是握法、不跨步是发力顺序、用胸接是接球方式。它们的共同点是「某一环断了」，而环是可以一条条接上的。力气小的人把链接顺，照样扔得又远又稳。）
  • 误解：纠错就得一次把所有毛病一起改掉。 （澄清：正好相反。一次盯一环最有效——widget 会告诉你哪一环最拖后腿，先修那一个。链是一环环接顺的，同时改五样只会手忙脚乱、哪样都没练好。）
  • 误解：球翻跟头是球的问题 / 球太怪，换个球就好了。 （澄清：翻跟头几乎总是握法与出手的问题——满把攥死、食指没拨鞋带（回 01）。同一颗球，握对了、食指最后一勾，立刻就转起来。别怪球，查你的手。）
  • 误解：动作对不对，光靠「感觉」就能判断，不用刻意自查。 （澄清：很多错恰恰是「感觉不到」的——比如提前抬头漏接，你当时根本没意识到眼睛离开了球。所以每条毛病都配了能当场验证的自查信号（看脚、摸哪里酸、问自己有没有看见球进手），靠这个，而不是靠模糊的感觉。）
  • 误解：接不住球，是我手不够快 / 反应慢。 （澄清：绝大多数漏接不是快慢问题，而是方法问题——用胸挡（不能卸力）或提前抬头（眼睛离球）。修的是方法：用手接、看球入手，而不是去练什么反应速度。）
  • 误解：和同伴总对不上，是对方扔得 / 跑得不好，跟我没关系。 （澄清：配合是两个人共同的时空，光怪一方没用。要分清是「球到人没到」还是「人到球没到」，再一起约好步数、时机、提前量（回 05）——这是双方的事，也最能靠沟通快速修好。）
