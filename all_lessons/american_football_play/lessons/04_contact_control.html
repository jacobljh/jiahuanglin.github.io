第二部分 · 传与接（把球送出去、稳稳收回来）

接球：用手不用胸，看球入手再收好

你已经能把球又快又稳地掷出去了（03）——可传球是两个人的事。球在空中划着漂亮的螺旋飞过来，然后呢？绝大多数新手的本能是用身体去挡：手往两边一张，胸口一迎，想拿身体当块大墙把球「兜」住。结果几乎每次都一样——球「咚」地撞在胸口，弹飞了。问题又是那个老毛病的镜像：这一次你不是「只用胳膊甩」，而是干脆不用手、拿最硬又最不会「收力」的胸口去接。这一课把接球拆成三件必然的事：用手不用胸、看球入手（look it in）、接住再收好（tuck）。你会发现，接球从来不是运气，而是手型对不对、有没有看球到底、收没收好这三步推下来的结果。

本课路线
(1) 先问「接球的任务到底是什么」——不是「碰到球」，而是让飞来的球停在一个最可控、能马上护住带走的地方；由这个任务反推出第一条铁律：用手接、不用胸挡（因为手能主动「收力」，胸口只会弹）；(2) 第二件事——看球入手：眼睛得全程盯着球一路进到手心，多数漏接是「没看到底、提前抬头想着跑」；(3) 第三件事——手型得随来球高度变：高球拇指食指搭成三角窗口、低球小指并拢兜成篮子；(4) 第四件事——接住只是开始，得立刻收好（tuck）把球夹进 high-and-tight，顺序不能反；(5) 亲手拨一个部件，眼看着手型是否匹配、有没有看球，怎样把接球成功率从「一定弹飞」拉到「稳稳收下」；最后交棒第 05 课——你一个人能接了，可传接是两个人的默契，球得送到你将要到的地方，于是「跑位与配合」就成了下一个绕不开的问题。
数据小注
本课 widget 里出现的「接球成功率」百分比，是示意 / 量级值，用来帮你建立直觉——它想让你抓住的是「手型对上来球高度、且看球入手，成功率就高；用胸接、手型接反、或不看球，成功率就骤降」这个方向与对比，不是某个人在某场比赛里的实测命中率。真实接球成功率因人、传球质量、防守压力差异极大。请记住那个「拨对了就稳、拨错了就弹」的对照，别记具体数字。

一、先问任务：接球到底要「接」成什么样？

先把上一课接住：你（作为传球手）已经能把球又快又稳地掷出去了。现在换个位置，你是接球的人（receiver）。球在空中朝你飞来，你要干什么？

新手几乎都会把任务想成「碰到球、别让它掉地上」——于是本能地把身体迎上去，手臂大张、胸口一挺，像用一面墙去截一个飞来的东西。这个直觉错在哪？错在它把任务定小了、也定歪了。接球真正的任务，从来不是「碰到球」那一下，而是——

接球的真正任务
让飞来的球，停在一个你「最可控」的地方：既接得牢（不弹、不脱手），又能立刻护住、马上带着它跑。换句话说，接球不是一个「挡」的动作，是一个「柔和地把飞来的动量卸掉、再把球收进你携球链子的起点」的动作。这句话就是本课所有手法的总源头：用什么接（手，不是胸）、怎么看（盯球入手）、怎么摆手（随来球高度变手型）、接住后干什么（立刻收好），全是为了完成「可控地卸力 + 马上能护住」这一个任务。

你看，一旦把任务从「碰到球」升级成「让球停在最可控、能马上带走的地方」，后面每一步就都不用背了——它们会像结论一样被逼出来。第一个被逼出来的，就是那条最反直觉、却最关键的铁律：用手接，别用胸挡。

二、用手不用胸：因为手会「收力」，胸口只会弹

为什么怕漏球、想拿胸口去兜，反而一定弹飞？这里藏着接球最核心的一点力学。球飞过来，是带着动量的——它有速度、要往前冲。你要接住它，本质上就是要在很短的时间里，把这股向前的动量卸掉、让它相对你「停下来」。关键就在怎么卸这股力。

胸口是硬的、而且不会主动后退。球撞上一堵不会退让的硬墙，卸力的时间极短、冲击极猛，于是球像皮球撞墙一样「咚」地被弹开——你的胸口非但没帮你接住球，反而给了它一个反方向的力，把它弹飞。这就是「用身体挡一定弹」的全部原因。

而手不一样：手能主动「收力」。球一触到手指、手掌，你就顺着球飞来的方向把手往后带一点——让接触的过程延长，用更长的时间、更柔的路径把动量一点点吃掉。就像接一个鸡蛋：你绝不会把手僵在那儿等鸡蛋砸上来（那准碎），你会在鸡蛋落进手心的一瞬间顺势往下沉，把冲击「化」开。接球是一模一样的道理。于是这条铁律被推了出来——

于是：手法被推导出来
用手指和手掌去接，别用胸；两只手要在球前面组成一个「网」；手臂微屈、放松，球一触手就顺着球飞来的方向随球后收，把冲击柔和地卸掉。手是你全身最灵巧、最能「随球让步」的末端——正因为它软、能退、能收，它才接得住；胸口最硬、最不会退，所以它只会弹。越怕漏、越想拿身体去堵，就越弹得凶。

这里有个和全课主线呼应的漂亮对称：掷球时（03），你怕的是「只用胳膊」——末端太弱、发不出力；接球时，你怕的却是「不用手、用胸」——放着最灵巧的末端不用、拿最硬的躯干去硬碰。发和接是两件相反的事，但手这个灵巧的末端在两头都是主角：发球靠它最后一拨把力和旋转送出去，接球靠它最先一触把动量柔和地收回来。

三、看球入手（look it in）：眼睛得跟球到手心

手型再对、卸力再会，如果手没在正确的时刻、出现在正确的位置，照样接不住。那手怎么知道该在哪、该什么时候合拢？答案朴素得让人意外：靠眼睛，一路盯着球看到它进手。

这就是橄榄球里反复被念叨的一句话——look it in，「把球看进手里」。它的力学再直白不过：你的手要在对的时刻、对的位置合拢，唯一可靠的信息来源就是眼睛对球的实时追踪；球在飞行末段其实又快又会飘，你只有盯到它真正落进手心的那一刻，手的时机才能对上。

新手病：提前抬头
绝大多数漏接，根子都是同一个动作——还没接住，眼睛就先抬起来了。为什么会抬头？因为你太想着「接住之后赶紧跑」，脑子已经飞到下一步，眼睛于是提前离开球、去看跑道、看防守。可就在你抬头那零点几秒，手失去了对球的实时信息，时机和位置全乱——球从指缝里溜走、或砸在手掌根上弹开。你不是手不行，是眼睛提前叛逃了。

于是：动作被推导出来
目光全程跟着球，一路盯到它落进你的手心、被手指合拢收住为止——先把球看进手里、收好，再抬头看往哪跑。记住那个顺序：看球 → 接住 → 收好 → 再抬头，绝不能反过来。眼睛在球上多待那零点几秒，比什么手上技巧都管用。

四、高球用三角：拇指食指搭成一个窗口

好，用手、会收力、盯着球——但「手到底怎么摆」，还得看球是从高处来还是从低处来。这不是两套要背的口诀，而是同一个道理在两种高度下的两个解：你要让两只手在球前面组成一个稳稳的「网」，而这个网的形状，得让掌心尽量朝着球、十指能兜住球、且不留能让球钻过去的大缝。来球高度不同，能满足这三条的手型自然就不同。

先看高球——球飞到胸口以上，包括正前方高位、需要你把手举到脸前或头前去接的球。这时候你的两只手在身前一举，最顺手、最能兜住球的摆法是：两个拇指相对、两根食指相接，让拇指和食指搭出一个三角形（triangle）的窗口，掌心朝向来球。球就接进这个三角窗口里，被十指从四周合拢收住。

高球手型 · 三角（triangle）
两手举在球前，两拇指相对、两食指相接，拇指与食指搭成一个三角形的窗口，掌心朝球；把球接进三角里，再十指合拢、随球后收卸力。为什么是三角？因为球从高处来，掌心必须朝上前方去迎它——只有拇指在下、食指在上搭成这个三角，你的两只手才能掌心朝球、又在球正前方合成一张没有大缝的网。这是「胸口以上接球」这个任务唯一顺手的手型。

五、低球用兜：小指并拢，掌心朝上成一个篮子

再看低球——球飞得低，落在你腰部以下，得你把手放低、甚至弯下腰去接的球。这时候如果你还想用高球那个「拇指在下」的三角，手腕会拧得极别扭、掌心根本朝不到球。任务变了（球从低处来），顺手的解自然也就变了。

低球的解是：两只手小指并拢、掌心朝上，像捧起一个「兜」或「篮子（basket）」，把球从下面兜住（两个拇指在外侧）。掌心朝上、小指相靠，你的两只手就在球下方合成一个向上的篮子，球落进来正好被兜住、再顺势往身上收。

低球手型 · 兜 / 篮子（basket）
两手小指并拢、掌心朝上，组成一个向上兜的篮子（拇指在外侧），把低球从下方兜住，再随球往身上收好。为什么反过来、小指并拢？因为球从低处来，你得从下面把它兜住、掌心必须朝上——只有小指相靠、拇指在外，两只手才能在球下方合成一个没有漏底的篮子。它和高球的三角，是同一条「掌心朝球、合成一张没缝的网」原则，在高、低两种来球下的两个镜像答案。

高球（胸口以上）→ 三角
两手举在前，两拇指相对、两食指相接，搭成一个三角形窗口，掌心朝球，把球接进三角里。
口诀：拇指食指搭三角，掌心朝球。
低球（腰部以下）→ 兜 / 篮子
两手放低，小指并拢、掌心朝上，拇指在外，捧成一个向上的篮子，把球从下面兜住。
口诀：小指并拢朝天兜。

（还有一种：球飞到你身体两侧、不高不低时，转身、用手主动迎上去接——但你先把「高用三角、低用兜」这两种练成本能，绝大多数球就都有解了。）一句话记住这两个手型的分界：球比胸高，拇指食指搭三角；球比腰低，小指并拢朝天兜。拨反了——比如拿三角去接一个膝盖高的低球，或拿小指并拢的兜去接一个脸前的高球——手腕别扭、网也合不拢，球十有八九弹开。

六、接住再收好（tuck）：先收好，再抬头

球稳稳落进手里了——大功告成？还差最关键的一步。别忘了这是橄榄球：你手里这颗球，防守方随时会来拍、来掏。刚接住的球如果还松松地端在身前，一个巴掌拍过来就掉了——接住只是第一步，球随时会被打掉。

力学上这一步顺理成章：接住的那一瞬间，球还在你最暴露、最容易被打掉的位置（身前、双手捧着）；要让它安全，就得立刻把它挪到最难被夺走的地方。这个地方你在 02 已经学过了——high-and-tight：球的一头顶在肘弯 / 前臂内侧，另一头被手掌盖住，整颗球夹进腋下、贴紧躯干，用四个接触点把它锁死。所以接球的最后一步，就是把刚接住的球立刻收进这个 high-and-tight——这个动作橄榄球里叫 tuck（把球「掖」进去）。

于是：tuck 的顺序被定死
接住 → 立刻把球夹进腋下 / 前臂肘弯（收进 02 学的 high-and-tight）→ 再抬头看往哪跑。这个顺序绝不能反：不是「先抬头找路、再顺手收球」，而是先把球收好，再抬头。这和第三节「看球入手」是同一条纪律的两面——它们都在对治新手那个「太急着跑」的本能：你越急着看下一步，就越会在球还没安顿好的时候抬头、松手，于是漏接或被打掉。把球收进腋下锁死，多花那零点几秒，比任何花哨跑动都值。

七、动手：手型 + 看球，决定你接不接得住

光说「高用三角、低用兜、还要看球入手」还太抽象。下面这台小玩具让你亲眼看到：同一颗飞来的球，就因为你选的手型对不对、有没有看球入手，结果会是「稳稳收下」还是「砸胸弹飞」。你要做三件事：(a) 选来球高度（高 / 中 / 低）；(b) 选手型（三角 / 小指并拢的兜 / 用胸接）；(c) 开关「看球入手」。然后按「接一球」，看那颗球飞进来，按你当前的设置被接住、还是弹开。

去把它拨明白：给一个高球配三角、给一个低球配兜，再把看球入手打开——右侧的「接球成功率」会一下子拉到很高，球稳稳落进手里、收进腋下。然后故意拨错：拿兜去接高球、或干脆选用胸接、或把看球关掉——你会看到成功率骤降，动画里球「咚」地弹开。你会得出和全课一致的那个结论：接球不是运气，是「手型匹配来球 + 看球入手」这两件事凑齐了的必然结果。

八、把这一课接回主线

一句话带走
接球，是把「飞来的球的动量」用手这个最灵巧的末端柔和地卸下来、再把球收进你携球链子的起点。所以：用手接、不用胸（手能随球后收卸力，胸口只会弹）；看球入手（眼睛盯到球落进手心再抬头，多数漏接都是提前抬头）；手型随来球高度变——高球两拇指两食指搭成三角、低球小指并拢兜成篮子；接住后立刻收好（tuck）把球夹进腋下的 high-and-tight，先收好再抬头，顺序绝不能反。接球不是运气，是这几件事推下来的结果。它和 03 的掷球是一枚硬币的两面：发球靠手指最后一拨把力送出去，接球靠手掌最先一触把动量收回来——灵巧的末端，两头都是主角。

你现在一个人能接住球了。但别忘了本课开头那句话：传球是两个人的事。刚才 widget 里的球是「朝你飞来」的——可真到场上，传球手往哪扔？如果他对着你现在站的地方扔，等球飞到，你早跑走了，球只会落在你身后。真正的传接，传球手必须把球送到你将要到达的位置（lead the receiver，「把球送到你前面」）；而你也得沿着一条两人事先说好的路线（route）、在对的步数、对的时机干脆利落地跑到那个点。于是下一个绕不开的问题，就是把 00–04 学的这些单项技术，第一次串成两个人的默契——跑位与配合。

常见误解

  • 误解：怕漏球，用身体（胸口）挡最保险。 （澄清：正好相反，用胸一定弹。胸口是硬的、不会退让，球撞上去卸力时间极短，像皮球撞墙一样被弹开。要用手——手能随球飞来的方向后收，把冲击柔和地卸掉，像接鸡蛋一样。越怕漏、越想拿身体堵，弹得越凶。）
  • 误解：接住球最关键的是手快、手感好。 （澄清：更关键的是眼睛——看球入手（look it in）。绝大多数漏接是因为球还没进手、眼睛就先抬起来想着跑了，手因此失去时机。目光盯到球落进手心、收好，再抬头，比任何手上技巧都管用。）
  • 误解：不管球高球低，两只手举一样的手型接就行。 （澄清：手型必须随来球高度变。高球（胸口以上）要两拇指相对、两食指相接搭成三角，掌心朝球；低球（腰部以下）要小指并拢、掌心朝上兜成篮子。用错了手腕会别扭、网合不拢，球必弹。）
  • 误解：接高球也用「小指并拢」那套（或接低球用「拇指并拢」的三角）。 （澄清：接反了。小指并拢的兜是给低球的——从下面兜住；拇指食指的三角是给高球的——掌心朝上前方去迎。拿兜去接脸前的高球、或拿三角去接膝盖高的低球，掌心都朝不到球，十有八九弹开。记住：球比胸高搭三角，球比腰低朝天兜。）
  • 误解：球接住了就赶紧抬头看往哪跑，收球晚一点没关系。 （澄清：顺序反了。刚接住的球在身前最容易被拍掉、掏走，必须先收好（tuck）——立刻夹进腋下、收进 02 学的 high-and-tight——再抬头。先抬头再收球，正是丢球的高发时刻。看球入手和先收后抬头，是同一条「别太急着跑」的纪律。）
  • 误解：接球是天生的手感 / 运气，练不出来。 （澄清：接球是可推导、可练的：用手不用胸 + 手型匹配来球高度 + 看球入手 + 接住收好，这四件事凑齐了，接住就是必然结果。widget 里你能亲眼看到——拨对了绿灯稳收、拨错了红灯弹飞，一点都不玄。）
下一步
你一个人能把球稳稳接下、收好了——可传接是两个人的事：球得送到你将要到的地方，你也得沿一条说好的路线在对的时机跑到那个点。 → 第 05 课《跑位与配合：路线、时机与提前量》会带你把 00–04 的单项技术第一次串成两人的默契：基础跑位路线（out / slant / go）怎么跑、为什么要在固定步数后干脆切；触身 / 腰旗最精简的规则；以及最关键的一点——传球手要把球送到接球人前面（lead the receiver），而不是他现在站的地方。你会发现，「提前量」也不是规定，是被「两个都在动的人要在同一个点、同一个时刻碰上球」这个任务逼出来的唯一解。
