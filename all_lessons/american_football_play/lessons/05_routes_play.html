第二部分 · 传与接（把球送出去、稳稳收回来）

跑位与配合：路线、时机与提前量

你会掷了（03），也会接了（04）——可到现在，这两件事还是你一个人的功夫。真正的传接，是两个人的默契：接球人往哪儿跑、什么时候切、传球人把球扔到哪儿，才能让球和人刚好在同一个点、同一个时刻碰上？这一课，我们第一次把前几课练出来的「好传球、好接球」放进两个人共享的时空里。你会学到三条最基础的跑位路线（out / slant / go）、什么叫「准时」，以及本课的灵魂——提前量（lead the receiver）：球飞过去要时间，而人一直在跑，所以传球人绝不能把球扔到接球人现在站的地方，而要扔到他将要到的那个点。最后再给你一套最精简、够玩起来的触身 / 腰旗规则。

本课路线
(1) 先问「为什么要跑约定好的路线（route）、而不是随便乱跑」——因为传球人得能预判你在哪、好提前把球送出去；(2) 给出三条基础路线 out / slant / go，讲清各自往哪跑、什么时候用、切的时候为什么要干脆利落（呼应 02 的降重心变向）；(3) 说清时机（timing）——路线的价值全在「准时」，QB 常在你要切的那一刻球就已经出手；(4) 推出本课灵魂提前量 lead：任务=球和奔跑中的人相遇→于是必须把球送到他将要到的点，这是一个躲不掉的推论；(5) 一套够玩就行的触身 / 腰旗极简规则（并强调规则各局可自定、先约好）；(6) 亲手拨滑块——选一条路线、拨一个提前量，按「开球」看球和人到底能不能对上。最后交棒第 06 课：把 00–05 学到的每个动作，逐一还原成「动力链或配合断在哪一环」。
数据小注
本课出现的码数、跑动步数、飞行时间等一律是示意 / 量级，帮你建立「球飞要时间、人一直在跑、所以必须打提前量」这个直觉，不是某场比赛的实测。widget 里「差 2 码 / 3 码」这类数字也是示意——请记住那个「lead 太小球落人身后、太大够不着、适中正好相遇」的对比，别记具体数字。

一、为什么要跑「路线」，而不是乱跑

想象你和一个伙伴玩传接：他拿球，你往前跑，等他扔。如果你随心所欲地乱跑——一会儿往左、一会儿回头、一会儿又冲深——会发生什么？他根本不知道你下一步在哪。而传球这件事有个躲不掉的物理事实：球飞出去要时间。等他看清你停在哪、再瞄准、再出手，球飞到时，你早已经跑到别处去了。于是要么他不敢扔（等你停下来，可你一停就被防守贴死了），要么扔出去也对不上。乱跑，把「两个人配合」这件事变成了不可能完成的任务。

所以答案只有一个：你们得事先约好你怎么跑。这条事先约好的跑动轨迹，就叫路线（route）。它是你和 QB 之间的暗号——你按约定跑，他就能预判你几秒后会出现在哪儿，从而提前把球往那个点送。任务（让球和人对上）→ 力学事实（球飞要时间、人在动）→ 于是你必须跑得让对方可预测：这就把「路线」逼了出来。它不是教练的规矩，是「两个人想在时空里对上一个点」的必然前提。

一句话抓住它
路线 = 你和传球人共享的一张「时间表」。你按它跑，他就知道「第几步、你会在哪」，才敢在球到达之前就把球扔向那个点。乱跑=撕掉时间表，配合当场崩溃。

二、三条基础路线：out / slant / go

路线可以有几十种，但零基础休闲局，你只要先会三条就能玩得很痛快。它们的差别，全在往哪个方向切、什么时候用。看下面三张俯视示意（球都从下往上、朝对方球门方向推进）：

out（外切 / 短横）

先直线冲几步（示意 ≈5 码），然后急停、90° 朝边线切出去。

怎么用：快速、稳当的短球。你切向边线、把防守甩在里侧，接到球顺势一步就能踩出界停表或保住球。
要点：那个 90° 转弯必须干脆——先降重心再蹬地（这正是 02 学的变向），否则拖泥带水，时机全乱。

slant（斜切）

起步后斜向内侧约 45° 切进空当，几乎马上就能接到球。

怎么用：最快能出手的路线之一。防守站得靠外时，你一斜插进中路的空档，QB 几乎立刻就能把球塞给你。
要点：切进去的那一下要果断、线路要笔直，别拖成一条软软的弧线——弧线会让「你几步后在哪」变得难猜。

go / fly（直线深传）

一路直线、全速冲深，比谁跑得过防守。

怎么用：搏长传、搏一击致命。你只管头也不回地全速跑，QB 把球高高送到你前方远处，你去「跑进球里」。
要点：这条路线对提前量最敏感——你在高速跑，球又飞得远，QB 扔到你当前位置必然落在你身后。它是理解下一节的最好例子。

三条路线，一个共同的动作要求：该切的时候要「切得干脆（sharp cut）」。为什么？因为路线的全部价值，是让 QB 能掐着时间预判你在哪。如果你切得拖沓、圆滑，你「到达转折点」的时间就变得模糊，QB 也就没法在对的时刻出手——整条时间表就糊了。干脆的切，既能真的把防守甩开（急停急转，对方跟不上），又能让你准时出现在约定的点上。这就是 02 教的「先降重心、再蹬地变向」在配合里的直接价值。

三、时机：路线的价值，全在「准时」

这里要点破一个新手常常想不到的事实：很多时候，QB 在你还没切、甚至还没到位的那一刻，就已经把球扔出去了。他不是等看到你空出来才扔——那样太慢，球到时防守早补上了。他是相信你会准时跑到约定的点，于是在你即将转身的那一刻提前出手，让球「在你转过身、抬头找球的瞬间」正好到。

这就解释了为什么时机（timing）和路线是一体两面的：路线约定了「你会去哪」，时机约定了「你几时到那儿」。两者都对上，配合才成立。所以你跑路线时，脑子里要有个节奏：按约定的步数跑、在约定的那一步果断地切——早了、晚了、拖了，球就会扑空，哪怕你路线跑得再标准。准时，是路线的灵魂；干脆的切，是准时的保证。

把前几课接上来
注意这条链是怎么串起来的：03 让你能把球又快又稳地送出去，04 让你能把球稳稳收住，02 让你能干脆地变向——而这一课，是把这三样放进两个人的同一张时间表里：你干脆地切（02）、准时到点、QB 把 03 练出的好球提前送到，你再用 04 的手法接住。单人的动力链产出，第一次变成了两个人的配合。

四、本课灵魂：提前量（lead the receiver）

现在推导这一课最重要的一件事。它不是技巧，是一个躲不掉的物理推论，我们一步步逼出来：

1任务：让球和正在奔跑的接球人，在同一个点、同一个时刻相遇。
2两个事实：(a) 球飞过去要花时间（不是瞬间到）；(b) 在这段时间里，接球人一直在往前跑、没停。
3于是必然：如果 QB 把球扔向接球人现在站的位置，等球飞到，人早就跑过去了——球会落在他身后。所以 QB 必须把球扔到接球人将要到达的那个点（身前一步、几步，视他跑多快、球飞多久而定）。
4接球人这边：你也别减速等球，要「跑进球里」——保持速度、朝那个约定的相遇点跑过去，把球接进怀里。

这个「把球送到他将要到的点」的动作，就叫提前量（lead the receiver），也叫「打提前量」。它是传球人一切远球、跑动球的核心。你可以用一个熟悉的画面理解它：就像用水管去浇一只跑动的小狗——你不会把水柱对准它此刻的位置（水过去时它已经跑开了），你会把水柱洒到它前面一点，让它自己跑进水里。传橄榄球一模一样。

提前量，一句话
人在跑、球要飞一会儿才到，所以把球扔到他「现在」的位置一定落在他身后——传球人必须瞄准他「将要到」的那个点。提前量太小＝球落在人身后（他跑过头了）；太大＝球飞到人前面，够不着；刚好＝球和人同时到达同一点，接到。跑得越快、传得越远（比如 go 路线），要留的提前量就越大。接球人则「跑进球里」，别停下来等。

五、够玩就行：触身 / 腰旗的极简规则

要真的两个人（几个人）玩起来，还得有一套最起码的规则。好消息是：休闲局的规则非常灵活，下面这套是「够玩起来」的最精简版本，够你和朋友在草地或公园立刻开打。

最精简一套 · 规则可自定、开打前先约好
先郑重说明：橄榄球（尤其休闲的触身 / 腰旗）规则各地、各局都不一样，没有唯一标准。下面只是最精简、能立刻玩起来的一套；真正开打前，请和大家把规则、场地大小、几档、能不能过线前传都口头约定清楚。
1不撞人、无护具。这是全系列的前提：擒抱不靠身体冲撞，而是触身（touch）=防守用手碰到持球人（通常一手 / 两手触及即算停）；或腰旗（flag）=拔下持球人腰间挂的旗子即算停。安全第一。
2从开球线（line of scrimmage）发球。进攻方在球所在的这条横线上开始一档（一次进攻）。球被拦停在哪儿，那儿就成为下一档的新起点（新的开球线）。
3几档内推进到约定线。示意：给进攻方 4 档，要在这几档内把球推进到事先约好的一条线（或直接推进到达阵区 end zone）。推进到了，得分或换新一组档数；没推到，攻守交换。
4传球：每档都能传。休闲规则常允许在开球线附近做一次前传（forward pass）。但过线能不能传、一档能传几次，各局约定不同——先约好。接住＝推进，没接住（落地）＝这一档作废、回原开球线打下一档。

看出来了吗？这套规则的每一条，都在为「跑位 + 传接」搭一个舞台：有了开球线，你才知道路线从哪儿起跑；有了档数和目标线，才有「这一下要推进多少」的任务；有了触身 / 腰旗这种不撞人的停球方式，零基础的你才能安全地跑起来、接起来。规则不是负担，是让你刚学的那些动作有地方施展。

六、动手：选一条路线，拨对提前量，看球和人能不能对上

光讲「要打提前量」还是抽象。下面这台小玩具让你亲手拨出那个道理。这是一小块场地的俯视图：底部是开球线，接球人（蓝）从开球线起跑，QB（黄）在后方传球。你来做三件事：

(1) 选一条路线（out / slant / go）——决定接球人怎么跑；(2) 拨提前量 lead 滑块——决定 QB 把球瞄向接球人身前多远（从「扔在他当前位置」一直到「扔到他前方过多」）；(3) 可选地拨接球人速度；然后按 ▶开球。接球人会沿路线跑，跑到约定点时 QB 出手，球以固定的飞行时间飞向瞄准点。看球和人是不是在同一个点相遇——右侧读数会告诉你接到 / 过头（球飞人前面）/ 够不着（球落人身后）、相遇误差（码，示意），以及一个最佳提前量提示。

你会拨出一个让人恍然大悟的规律：lead 太小，球落在人身后（他跑过去了）；lead 太大，球飞到人前面（他够不着）；只有适中，球和人才正好相遇、接到。而且你会发现：换成 go（全速冲深）或把速度调快，要留的提前量就得更大——因为人跑得越快、球飞得越久，他在球飞行途中「窜」出去的距离就越远。这，就是「必须把球送到他将要到的点」最直白的一张图。

跑位 + 提前量 · 球和人能不能在同一点相遇
先选路线（out 外切 / slant 斜切 / go 直线深传），再拖提前量 lead：往左＝QB 瞄准接球人当前位置（球到时人早跑过→落人身后），往右＝瞄向他前方越来越远（太远→球飞到人前面够不着）。可再调接球人速度。按 ▶开球播放：接球人跑到约定点、QB 出手、球飞固定时间到瞄准点，看是否相遇。右侧 结果 / 相遇误差 / 最佳提前量 实时更新。码数、飞行时间为示意/量级值，帮你抓「人在跑、必须把球送到他将要到的点」这个道理。

多选几条路线、多拨几次提前量、多按几次开球，那个贯穿本课的事实就会自己跳出来：没有哪一次是靠「扔到他现在站的地方」接到的——每一次接到，都是因为球被送到了他将要跑到的那个点。这就是「lead the receiver」最直白的一张图，也是把单人技术变成双人配合的那把钥匙。

七、把这一课接回主线

一句话带走
传接是两个人共享的一张时间表：路线让传球人能预判你在哪、准时地干脆切让你按点出现、而提前量（lead）是躲不掉的物理推论——球飞要时间、人一直在跑，所以球必须扔到他将要到的点，不是他现在的点；接球人则「跑进球里」。这一课把 02（干脆变向）、03（好传球）、04（稳接球）第一次拧成两个人的配合。触身 / 腰旗给它一个安全、不撞人的舞台，规则各局可自定、开打前先约好。

回到全系列的引擎：前几课，你在打磨一个人的动力链——让力从跨步转髋一路传到手指、把球又快又稳地送出或收住。这一课，你把这条链的产出（好球、稳接）放进了两个人共享的时空里：你负责准时地把身体送到相遇点，QB 负责把球提前送到那里。到此，00–05 的每一块技术都齐了——握球、站姿、变向、掷、接、跑位配合。可是练归练，一到真跑起来，坏习惯就会冒出来：满握没螺旋、忘了跨步、用胸接球、切得拖沓、传球没提前量……它们各自断在动力链或配合的哪一环？该怎么自查、怎么纠正？这正是下一课要做的事。

常见误解

  • 误解：跑位就是拼命往前跑、往空地跑，看 QB 什么时候扔就行。 （澄清：那是乱跑。传球人根本没法预判乱跑的你在哪；而球飞要时间，等他看清你停哪儿再扔，你早跑开了。你必须跑约定好的路线、还要回头 / 抬头找球，让两个人对上同一张时间表。）
  • 误解：切的时候圆滑一点、跑成弧线更省力、也不容易摔。 （澄清：拖沓、圆滑的切会毁掉时机——你「几步后到哪」变得模糊，QB 就没法在对的一刻出手。要干脆利落地切（sharp cut）：先降重心再蹬地变向（02 学的），既甩得开防守，又准时到点。）
  • 误解：传球就该瞄准接球人、把球稳稳扔到他身上。 （澄清：他一直在跑，球飞过去要时间——瞄准他现在的位置，球到时必然落在他身后。必须打提前量，把球送到他将要到的点；跑得越快、传得越远，提前量越大。）
  • 误解：接球人应该减速、停下来等球飞过来。 （澄清：你一停，防守立刻贴死你，而且你和「提前量瞄准的相遇点」也错位了。正确做法是保持速度「跑进球里」——朝那个约定的相遇点跑过去把球接住。）
  • 误解：没有约定也能玩，两个人随便传随便接。 （澄清：没有共享的路线和时机，配合本质上不成立——要么不敢扔、要么对不上。哪怕最随意的公园局，也得先约好你跑哪条线、他往哪儿扔。约定，是配合的前提，不是麻烦。）
  • 误解：玩橄榄球总得撞人、抢球，零基础太危险。 （澄清：这门课全程是不撞人、无护具的触身 / 腰旗版本——用手一碰或扯下腰旗就算停，没有身体冲撞。安全、友好，零基础也能立刻上场。规则还能自定，开打前约好即可。）
