第二部分 · 怎么制造一次好出手

传球与球的移动：把局部优势放大成空位

上一课你学会了用五个人的站位「拉开空间」，逼防守顾此失彼，撬出一块块半空地。可问题马上跟来：撬出半块空地，怎么把它变成一个真正没人防的大空位？让持球人自己运过去？人比球慢——等你运到，补防早回来了。答案朴素得不像战术：把球传过去。但传球远不止「挪动一颗球」。球比人快：一次突破吸来协防，再一脚传出去，防守就得整体轮转去追；而追球的防线，永远慢半拍。最初的那点局部优势，就这样像滚雪球一样，被一串传导从一侧「跑」到防守补不到的另一侧，最后落成一个谁也防不住的空位。

本课路线
(1) 先立起本课的物理常数——球比人快：传一次球只要零点几秒、几乎瞬间到位，而一个人横移到补防位置要一两秒，这点时间差是全部魔法的来源；(2) 拆解「突分(drive-and-kick)」这台放大器——突破吸引第二个防守者协防 → 他原本盯的人空了 → 把球踢出去(kick out) → 防守轮转去补 → 又有人空了 → 再传一手(swing / extra pass)，优势滚雪球般放大；(3) 讲清「多一次传球(extra pass)」的哲学——宁可放掉一个有人防的好出手，换一个无人防的更好出手；(4) 点出它和足球是同一台引擎（球比人快、用传导调动防守）；(5) 动手玩「突分传导链」，亲手一脚脚把球传出去，看防守网每次都慢半拍、空位一格格变大、当前最佳出手的每回合期望分一路爬升。
数据小注
本课里所有的「期望分」「轮转滞后」「空位大小」都是示意 / 数量级，用来把直觉讲清楚，不是某一场比赛的精确测量。两个量级请记牢、它们会在 widget 里反复出现：一记无人防的底角三分大约 1.1～1.2 分/回合（命中率约 38%～40% × 3 分，参见第 03、05 课），而一次有人护框、被干扰的勉强出手常常只有 0.8 分/回合 上下。传导的全部意义，就是把同一次进攻从后者一步步「搬」到前者。

一、本课的物理常数：球比人快

先承认一件上一课偷偷绕过去的事。第 05 课我们把五个人摊开、逼防守不敢收缩，于是篮下和外线撬出了空当。可你有没有发现，那些空当总是离持球人很远、而且只闪现一瞬？挡拆吸来协防、突破撕开一道缝，制造的往往只是「局部多打少」的半个空位——比如你在右侧形成了 2 打 1，左底角那个人彻底没人管。问题是：球在右侧，那个大空位在左底角，隔着大半个球场。

如果让持球的人自己带球横穿过去，会发生什么？一个人从右侧运到左底角，得花一两秒；这一两秒里，防守早就整体平移、把那个空位重新补上了。人，是追不上自己制造的空位的。

但球可以。一记横穿半场的传球，飞行时间不过零点几秒，几乎是瞬间到位。这就是这一整课的物理常数，请把它刻下来：

传球时间 ≪ 防守者横移补位时间　⟹　球永远比补防的人先到

换句话说：当优势在一侧被制造出来、空位却在另一侧时，唯一来得及利用它的办法，是把球传过去，而不是把人运过去。球比人快这件事，听起来像废话，却是现代进攻一切「传导」战术的唯一地基。下面我们就看它怎么被一层层榨干。

二、突分(drive-and-kick)：把「吸引—踢出—轮转」做成一台放大器

现在把这个时间差变成一台能反复运转的机器。它的标准零件叫突分(drive-and-kick)，分四拍：

1突破(drive)——持球人往篮下杀。这是一记威胁：再不管他，就是一次篮下上篮（高期望分）。
2吸引(draw)——为了不让他上篮，第二个防守者必须离开自己盯的人，过来协防、补到突破路线上。这一刻，被他放下的那个人空了。
3踢出(kick out)——突破的人不强行上篮，而是把球踢给那个刚刚被放空的队友。球瞬间到位，那个人接到时是空位。
4轮转(rotate)——防守为了补这个新空位，又得有人扑出去；他一扑，他原本盯的人又空了。于是再传一手(swing)……

看出门道了吗？每一拍，进攻都在用「球比人快」逼防守做一次来不及的轮转。防守不是不补——他们一直在补，只是永远慢半拍：球已经到了下一个点，补防的人还在扑上一个点的路上。第 05 课那块「半个空位」（防守者只是放空、还没真正离开），就这样在传导中被升级成「整个空位」（防守者已经扑向别处、彻底回不来了）。

这正是上一课那台「二选一」引擎的延续，只是从两个人放大到了五个人。挡拆逼一个防守者在「换防 / 放投」里二选一（第 04 课）；突分则逼整条防线在每一次传球后都被迫二选一——补这个新空位、还是守住手里这个人？而每做一次选择，就在别处漏出一个新洞。优势像滚雪球：第一次突破撬动的可能只是一道缝（期望分从 0.8 抬到 0.95 上下），可一脚 kick out、再一脚 swing 之后，它能滚成一记彻底无人防的底角三分（≈ 1.1～1.2 分/回合）。

三、「多一次传球」：宁可放掉好出手，换一个更好的

这台放大器引出了现代篮球里一句几乎成了信条的话——「多一次传球(extra pass)」。它说的是：当你手里已经有一个还不错、但有人防的出手时，如果再传一手就能把球送到一个无人防的更好出手，那就别投，传出去。

为什么这是对的？把它翻译成上一部分的货币就一目了然。假设你突破后接球，面前是一记有人扑防的三分——防守者正张手扑来，这球大概值 0.95 分/回合。看着不错，可以投。但你眼角瞥见：防守为了扑你，左底角那个人被彻底放空了。如果你再传一手过去，那将是一记无人防的底角三分——≈ 1.15 分/回合。

把它算清楚
这是一道纯粹的期望分比较题：0.95（有人防的好出手） < 1.15（无人防的更好出手）。放掉自己手里那个「好」球，去成全队友那个「更好」的球，每回合多榨出约 0.2 分。一场球几十个回合累加起来，就是稳稳几分的差距。「无私」在这里根本不是道德——它是算术。多传那一手球，是在主动把出手从期望分较低的点，挪到期望分更高的点。

当然，「多一次传球」有它的边界，诚实地说清楚：传球不是免费的。每多传一手，都多一次失误风险，也多耗几秒那本就稀缺的 24 秒（第 01 课）。所以信条的完整版本是：当且仅当下一手能把期望分抬高、且传球风险可控时，才多传。为了传而传、把一个空位三分倒成 24 秒违例，就走过头了。真正的高手，是在「这球够好了、该投」和「再传一手会更好」之间，每个回合做一次冷静的期望分判断。

四、和足球是同一台引擎

如果你读过姊妹轨道《足球的逻辑》，这一课会有强烈的既视感——因为它讲的是同一台引擎。足球第 05 课《传球与控球》里那句话几乎可以一字不改地搬过来：传球的真正目的不是位移皮球，而是「移动对手」。tiki-taka 用一连串短传逼对方防线不停平移、追球，直到某一脚把人拉出位、暴露出身后那块空当；篮球的突分传导，做的是一模一样的事——用球的快速移动，逼防守不停轮转，直到某一次他们追不回来、漏出一个空位。

两项运动在表面上天差地别（足球进球极稀有、篮球几乎每回合都在得分），可一旦深入到「怎么制造一次好机会」，它们收敛到了同一条原理：球比人快，所以用传导调动防守，把局部优势搬运、放大到防守补不到的地方。区别只在尺度——足球的场地大、人少、一次成功的传导调动可能直接换来唯一的进球；篮球的场地小、回合快、传导是每个回合都在跑的常规操作。但那个内核，是共享的（想看足球版怎么讲，去读 足球的逻辑 · 第 05 课《传球与控球：把球当作「移动对手」的工具》）。

五、动手：突分传导链，看防守每次都慢半拍

下面把前四节的逻辑做成一台可以亲手运转的小机器。屏幕上是一个半场，进攻方五个人按「五外」摊开，五个防守者各自贴人，篮下是一块开放的高亮区。一开始还没人动——这就是上一课结束时的局面：站位拉开了，但球还在原地，最好的出手只是一记有人干扰的勉强球（≈ 0.8 分/回合）。

你来拨动核心变量：点 下一手传导 ▶，依次走完 突破 → 踢出 → 横传(swing) → 多一手(extra pass)。每点一次，你会看到三件事同时发生：(a) 球（橙色）瞬间飞到新的点；(b) 防守网开始轮转去补，但每个轮转中的防守者都画在「上一拍该在的位置」——肉眼可见地慢半拍；(c) 当前接球点旁边那块空位一格格变大、变亮。盯住 KPI 里的每回合期望分：它会随着传导一路从 0.8 爬向 1.15——这就是「优势被传导放大」的数字证据。点 复位 回到静止。

玩两下你就抓住了这台机器的脾气：球每移动一次，防守就被迫轮转一次，而轮转永远滞后于球。静止时最好的出手只值 ≈ 0.8；突破吸来协防、kick out 之后抬到 ≈ 1.0；再一脚 swing、extra pass 把球甩到弱侧底角，防守彻底追不回来，那记无人防的底角三分值 ≈ 1.15。同一次进攻、同样五个人，仅仅因为让球多跑了几手，期望分就涨了将近 0.35 分/回合。这不是球员变强了，是球的移动替你把防守的「慢半拍」一次次兑现成了空位。

六、把这一课接回主线

退一步看，第二部分到这里走完了一条完整的链：第 04 课用挡拆逼一个防守者二选一；第 05 课用空间逼五个防守者都不敢收缩、撬出半空地；这一课用球的移动，把那半空地放大成真正的空位。三课其实在合力回答同一个问题——在有人防的情况下，怎么把一次出手送到第 03 课那张「期望分地图」上最值钱的格子里？答案是：拉开 → 撕开局部优势 → 用传球把优势搬运、放大，直到防守追不回来。

一句话带走
球比人快，这是全部魔法的来源：一次突破吸来协防 → 被放下的人空了 → 踢出去 → 防守轮转去补、却永远慢半拍 → 再传一手，于是别处又空了。最初的局部优势像滚雪球一样，被传导从一侧搬到防守补不到的另一侧，把第 05 课的「半个空位」放大成无人防的大空位。「多一次传球」不是道德，是算术：放掉一个有人防的好出手（≈ 0.95），换一个无人防的更好出手（≈ 1.15），每回合多榨 ≈ 0.2 分。这和足球的传导调动，是同一台引擎。
下一步
可有时候，球转了一整圈，对方防守纪律严密——每个人都在位、不咬饵、不漏人，传导就是走不通，没有空位。24 秒越走越少。这时把球塞给队里最能打的那个人，让他一对一单挑（「单打 iso」），常常招来一片「又不传球！」的骂声。可单打真的总是错的吗？ → 第 07 课《个人进攻：单打、造犯规与「什么时候单打是理性的」》会把单打从「自私」重新定义成一道期望分比较题：在错位、关键时刻、或面对顶级单打手时，它的期望分可能高于体系进攻能创造的。还会揭开单打最被低估的隐藏红利——造犯规：把对手送上罚球线，两罚 ≈ 1.54 分（回扣第 02 课），是全场最高效的得分之一。
