第三部分 · 防守的另一半

防守的目标：把对手每回合期望分压到最低

前面七课，我们一直站在进攻这一侧，把一台「期望分制造机」拆得透透的：一次回合的原子价值（第 01 课）、那把「命中率 × 分值」的乘法尺子（第 02 课）、铺满半场的期望分地图（第 03 课），再到挡拆、空间、传导、单打怎么抬高一次出手的期望分（第 04–07 课）。可这一切都只是硬币的进攻面。现在把镜头狠狠翻过去——对面那五个人，整整一回合都在拼命做一件和我们完全镜像的事。他们的目标函数是什么？答案简单到近乎残酷：把我们刚刚拼命抬高的那个期望分，给狠狠摁下去。

本课路线
(1) 先把防守的目标函数一句话立起来——最小化对手每回合期望分（PPP），它是进攻目标函数的镜像；(2) 用第 03 课那张期望分地图反推防守的优先级清单：① 死守篮下（最高价值区，护框第一）② 守住三分线、尤其底角（第二贵的区）③ 主动把对手逼向中距离（全场最便宜的区，让他投那儿反而是好事）④ 绝不送罚球（两罚 ≈ 1.54 分，全场效率最高的得分，回扣第 02、07 课）⑤ 不让进攻篮板（白送一个额外回合，预告第 10 课）；(3) 把这五条拢成「防守版四要素」的雏形（对应第 11 课）；(4) 拉一条横向对照——足球防守＝压缩高 xG 空间、棒球防守＝制造出局，三者讲的是同一件事：最小化对手的期望；(5) 动手玩「防守诱导」，亲手在三种防守策略间切换，看你把对手赶去了哪个区、他的 PPP 怎么随之涨落。

数据小注
本课所有命中率、各区每回合期望分都是示意 / 数量级，沿用第 03 课那张地图的口径，用来把逻辑讲清楚，不是某场某队的精确测量。供你做标尺的几个量级：篮下出手命中率约 ~60 % → 0.60 × 2 = 1.2 分/回合；底角三分约 ~38 % → 0.38 × 3 ≈ 1.1 分；弧顶三分约 ~35 % → ≈ 1.05 分；长两分 / 中距离约 ~40 % → 0.40 × 2 = 0.8 分，是全场最便宜的区；两次罚球 ≈ 2 × 0.77 = 1.54 分，是全场效率最高的得分之一。这些数字随赛季、球队、数据口径浮动，请当作「大致这么多」的标尺。

一、防守的目标函数：把进攻那句话，整句反过来念

这门课从第 01 课起就只有一台引擎：篮球是一门回合制的得分经济学。规则（24 秒时限、三秒区、三分线）把「每一次进攻回合」变成了稀缺资源，把每回合期望分（points per possession, PPP）变成了这项运动的货币。进攻方整整七课都在追问同一个问题——

进攻目标：最大化 我方每回合期望分（PPP）

那防守方在干什么？很多人会含糊地说「防守就是别让他得分」「拼抢、对抗、防住人」。这些都对，但都不是目标函数。把进攻那句话原封不动、整句反过来念，你就拿到了防守的全部纲领：

核心一拍
防守是进攻算术的镜像。进攻方要最大化自己的每回合期望分；防守方的唯一目标，就是把对手的每回合期望分最小化。防守目标：最小化 对手每回合期望分（对手 PPP）所有的防守动作——封盖、抢断、轮转、贴防、不犯规——都不是孤立的「拼劲」，而是为这一个数字服务的理性手段。看懂防守 = 看懂每一个防守动作在帮你把对手 PPP 压低多少。

这一拍是整个第三部分的地基。它把「防守」从一团关于意志和血性的模糊感觉，变成了一道和进攻完全对称的优化题。而既然目标是压低对手 PPP，那只要回到第 03 课那张地图，防守该干什么立刻就能被反推出来——不需要任何玄学。

二、反推防守优先级：把进攻的期望分地图，倒过来读

第 03 课我们铺过一张「期望分地图」：把那把「命中率 × 分值」的尺子贴满整片半场，每个区域都有一个 PPP 数值。对进攻方来说，这张图告诉他该往哪儿出手（去高价值区）。对防守方来说，同一张图反过来读，就是一张威胁排序表——对手在哪个区出手最伤你，你就该最优先掐死哪儿。

把第 03 课的几个地标按 PPP 从高到低排一遍，防守优先级自动浮现：

区域（进攻视角）每回合期望分（示意）防守该做什么（镜像）
篮下≈ 1.2 分（最高）① 护框第一：死守，这是最贵的区，绝不能轻易放进去
罚球线（两罚）≈ 1.54 分（最高）④ 绝不送罚球：无人防守的出手，比任何投篮都贵
底角三分≈ 1.1 分② 守住三分线、尤其底角：第二贵的区，最短的三分
弧顶三分≈ 1.05 分② 一并守住三分线（弧顶略低于底角，但仍是高价值）
中距离 / 长两分≈ 0.8 分（最低）③ 主动放：全场最便宜的区，把对手赶来这儿反而好

请盯住这张表里最反直觉的那一行：中距离。新手的本能是「哪儿都得防严」，仿佛把对手所有出手都顶到难看才叫好防守。但目标函数不答应——你只有五个人、两只手，资源是有限的，防住一个区就必然要在另一个区让出空隙。既然如此，理性的做法是把防守资源砸向最贵的区（篮下、三分线），同时故意把对手往最便宜的区（中距离）赶。让对手在中距离投得再准，每回合期望分也只有 ~0.8 分——那是在帮你压低他的 PPP。

这就是现代防守一句听起来像悖论、其实纯粹是算术的口号：「逼他投他最想投的、却最不该投的那个区。」下面把这张优先级表里的五条，逐条讲清它们各自在为压低对手 PPP 做什么。

三、第一优先级：护框——守住那个 1.2 分的篮下

篮下是全场出手命中率最高的区（~60 %），每回合期望分 ≈ 1.2 分，只低于罚球线。所以防守的第一块预算必须砸在这儿：护框（rim protection）。

护框压低对手 PPP 有两条路，缺一不可：

  • 降低篮下的命中率——一个能封盖、能垂直起跳干扰的护框者站在篮下，会把对手篮下那 ~60 % 直接拖下来好几个百分点。命中率一掉，那个区的 PPP 就跟着掉（期望分 = 命中率 × 分值，第 02 课）。
  • 降低篮下的出手数量——更狠的一招是让对手根本到不了篮下。一个有威慑力的护框者杵在那儿，进攻方会主动放弃突破、改投外线或中距离——他被劝退了。出手数从最贵的区挪走，本身就压低了整体 PPP。

这正是为什么哪怕在「人人投三分」的小球时代，一个顶级护框中锋依然珍贵：他守的是地图上最贵的那块地。护框者的威慑力，是整个防守体系能不能成立的地基——后面第 09 课讲的协防轮转，本质上都是在为「保住篮下」这件事服务。

四、第二优先级：守三分线，尤其是底角那条最短的三分

篮下守住之后，第二贵的区是三分线。回忆第 03 课的关键发现：三分不是「难度更高的两分」，而是规则用一条线切高了分值——同样的命中率，乘的是 3 不是 2。所以一记 ~38 % 的底角三分，期望分 ≈ 1.1 分，比一记 ~40 % 的中距离（~0.8 分）值钱得多。

而三分线上最该死守的，是底角。第 03 课讲过，底角三分是最短的三分（底线附近三分线离篮筐更近），命中率天然更高，期望分在所有三分里最高。对防守方来说，这意味着：

为什么底角是防守的「红色警报区」
底角三分 ≈ 1.1 分，逼近篮下的 1.2 分，却比篮下好出手得多（不用突破、不用对抗，站着投就行）。更要命的是，底角离篮筐近、留给防守者扑防的距离最短、轮转过去的路线也最远（你得从篮下一路冲到底线角落）。所以现代防守有一条铁律：宁可放弧顶，也别漏底角。一次没人防的底角三分，几乎是进攻方除了篮下和罚球之外能拿到的最高效出手——它是第 05 课「拉开空间」「需要会投三分的大个子」那套进攻设计，最想榨出来的终点。防守的任务，就是把这个终点死死焊住。

五、第三优先级（最反直觉的一条）：主动把对手逼向中距离

守完篮下和三分线，剩下的区是中距离 / 长两分——全场最便宜的区，~0.8 分。这里藏着现代防守最深、也最反「老派直觉」的一拍。

你的五个人是有限资源。把篮下和三分线两头都顶死，对手的球必然被挤压到中间那一圈无人地带。这时候防守方不该慌着再扑出去封中距离——恰恰相反，你应该松一口气，甚至暗暗高兴。因为：

对手被逼投中距离的期望分 ≈ 0.40 × 2 = 0.8 分 ＜ 篮下 1.2 ＜ 底角三分 1.1

对手在中距离投得再潇洒、再准，你每回合也只「输」~0.8 分，远低于让他投篮下或三分的代价。所以一套优秀的防守，会用站位、协防、堵突破路线，把对手主动赶进这个最便宜的区——这叫「放中距离（concede the mid-range）」。它不是防守失误，是故意的、理性的取舍：用最便宜的区，去换掉最贵的两个区。

这也正好和进攻那一侧严丝合缝地咬上：第 12 课会讲的「魔球（Moreyball）」，进攻方拼命消灭自己的中距离出手、只要篮下和三分；而防守方的镜像策略，就是反过来——拼命把对手逼回中距离。同一张期望分地图，进攻读出「别去哪儿」，防守读出「把他赶去哪儿」。这就是本课 widget 要让你亲手拨动的那个核心选择。

六、第四优先级：绝不送罚球——那是全场最贵的 1.54 分

现在轮到那条最容易被血性掩盖的铁律。回忆第 02、07 课反复敲打的那个数字：罚球线是球场上唯一没有任何防守的出手，NBA 罚球命中率常年 ~77 %，于是两次罚球的期望分 ≈ 2 × 0.77 = 1.54 分——比篮下（1.2）、比底角三分（1.1）都高，是全场效率最高的得分之一。

把这件事放进防守的目标函数里，结论冷酷而清晰：犯规，是防守方亲手把对手送进全场最贵的得分区。一次莽撞的封盖、一次重心已乱还伸手去掏的抢断、一次篮下的盲目对抗——哨一响，你不但没压低对手 PPP，反而把他直接拽上了那条 1.54 分的线。这是防守里负期望分的头号来源。

「好防守」和「脏活」的分界线，就画在这条数字上
新手常把「身体对抗凶猛、敢拼敢抢」等同于好防守。但目标函数只认一个数：这个动作让对手 PPP 涨了还是跌了？一次干净的垂直起跳封盖，压低对手篮下命中率 → 好；一次伸手犯规把人送上罚球线，≈ 1.54 分 → 极坏。所以顶级防守者的标志，往往不是封盖数，而是「用强大的威慑力影响对手出手，却几乎不犯规」——他们守纪律的「不动手」，比凶狠的「乱动手」值钱得多。

七、第五优先级：不让进攻篮板——别白送一个额外回合

前面四条都在压低单次出手的期望分。但防守方还有最后一件事必须做，否则前面全白费：把这次出手的篮板抢下来，结束这个回合。

回到第 01 课的底层定义——一个回合，止于「球权易手」。如果对手投丢了，你没把防守篮板保护住，让他抢到进攻篮板，会发生什么？这个回合没有结束，对手凭空获得了一次额外的进攻机会。你刚刚辛辛苦苦把他这次出手的 PPP 压到很低，结果他直接再来一次——你之前的防守努力被一笔勾销。

所以「保护防守篮板」是防守目标函数里不可省略的收尾动作：它守的不是某次出手，而是回合本身这个稀缺资源（第 01 课）。这条线索很重要，但它牵出的「回合数（pace）本身也是资源」「进攻篮板＝偷一个回合」是一整套独立的算术，我们留到第 10 课专门展开。这里只需记住：防守压低了出手 PPP 之后，必须用篮板把战果锁死。

八、把五条拢成一句：这就是「防守版四要素」的雏形

退一步看，上面五条优先级其实在反复指向同样几件事。如果把它们归并、去重，正好收敛成四条可量化的指标——这就是篮球数据分析里大名鼎鼎的「四要素（Four Factors）」的防守版（由 Dean Oliver 提出，我们留到第 11 课正式展开）：

① 限制对手的命中率（防守 eFG%）护框压低篮下命中、守住三分线（尤其底角）、把对手逼向低价值的中距离——三条优先级，殊途同归地都在压低对手「每次出手的期望分」。这是四要素里权重最大的一项。
② 逼对手失误（制造失误率）一次失误的期望分是 0 分——比任何出手都糟。逼出一次失误，等于把对手这一整个回合的 PPP 直接清零。这是我们这堂课没单独列、但同属「压低对手 PPP」的一条狠手段。
③ 保护防守篮板（防守篮板率）第七条：别让对手靠进攻篮板偷走一个额外回合。守的是回合本身（第 01 课），第 10 课细讲。
④ 不送罚球（控制对手罚球率）第六条：别把对手送上那条 1.54 分的线。犯规是防守里负期望分的头号来源。

看清楚这个收敛过程，你就理解了为什么「四要素」如此有力——它不是凭空发明的四个指标，而是「最小化对手 PPP」这一个目标函数，被那张期望分地图逼出来的四个抓手。第 11 课会把它和进攻版四要素并排放在一起，让你看清进攻与防守在数据上是如何严格对称的。

九、横向对照：足球、棒球的防守，讲的是同一件事

这台「最小化对手期望」的引擎，根本不是篮球独有。它是整个《球类的逻辑》系列的共同脊梁——每一种运动的防守，都是在最小化对手在该运动「货币」上的期望值：

运动进攻的「货币」防守 = 最小化对手的……手段
篮球（回合制）每回合期望分 PPP对手 PPP护框、守三分、逼中距离、不犯规、保篮板
足球（连续流动）期望进球 xG对手的 xG压缩高 xG 空间——把对手逼到角度小、距离远、人多的低质量射门位置
棒球（离散对决）得分期望 / wOBA对手的得分期望制造出局——每个出局都把对手的得分期望往零拉

三者表面天差地别，骨子里是同一道优化题。尤其和《足球的逻辑》第 10 课「防守：把空间压扁」对照着看会非常清楚：足球防守压缩的是高 xG 的空间（把射门逼到低概率的位置），篮球防守压缩的是高 PPP 的出手（把出手逼到低价值的中距离）——「空间」之于足球，正如「出手选择」之于篮球。而棒球防守则把这件事推到最离散的极端：直接制造出局，把对手的得分期望一个一个清零。看懂了篮球的「最小化对手 PPP」，你就同时拿到了读懂另外两项运动防守的钥匙。

十、动手：防守诱导——你把对手赶去了哪个区？

下面把全课的核心——「用有限的防守资源，把对手赶向哪个区」——做成可以亲手切换的实验。你有三种防守策略可选，每一种都对应「优先死守哪两个区、故意放掉哪个区」：

  • 放中距离（现代主流）——护框 + 死守三分线（尤其底角），把对手挤进最便宜的中距离。
  • 放底角（糟糕的取舍）——护框，但过度协防、漏掉了底角那记最贵的三分。
  • 放篮下（最糟的取舍）——死贴外线投手，却让对手轻松杀进最贵的篮下。

切换策略时，画面会重新分配对手在四个区（篮下 / 底角三分 / 弧顶三分 / 中距离）的出手占比，并按第 03 课那张地图的区值，实时算出对手这一回合的期望分（对手 PPP）。你的任务很简单：找出哪种策略能把对手 PPP 摁到最低——它会印证本课最反直觉的那一拍：主动放掉最便宜的中距离，才是把对手 PPP 压到最低的那条路。

多切几次你会看到一个干净的结论：三种策略放掉的区不同，对手 PPP 也跟着不同，而且高低顺序完全由那张期望分地图决定。「放篮下」让对手大量出手集中在 1.2 分的篮下，PPP 最高（最糟）；「放底角」把对手喂到 1.1 分的底角三分，PPP 次高；唯有「放中距离」把对手挤进 0.8 分的最便宜区，对手 PPP 被压到最低。同样是「放掉一个区」，放对了区，是顶级防守；放错了区，是灾难。防守的全部艺术，就是用有限的五个人，把对手赶向那张地图上最便宜的那块地。

十一、常见误解

  • 误解：好防守就是哪儿都防严、把对手所有出手都顶到难看。 （澄清：你只有五个人，防住一个区必然让出另一个区。理性的防守是有取舍地把资源砸向最贵的区（篮下、三分线），故意把对手赶向最便宜的中距离——目标是最小化对手 PPP，不是「处处都凶」。）
  • 误解：让对手投中距离是防守失败。 （澄清：恰恰相反。中距离是全场最便宜的区（~0.8 分），把对手逼去那儿是主动的、理性的取舍，能把对手 PPP 压到最低。这是「魔球」进攻逻辑的防守镜像。）
  • 误解：防守凶猛、敢抢敢拼、封盖多 = 好防守。 （澄清：目标函数只认一个数——这个动作让对手 PPP 涨了还是跌了。一次伸手犯规把人送上罚球线（≈ 1.54 分）是负期望分的头号来源。能用威慑力影响出手却几乎不犯规，比乱动手值钱得多。）
  • 误解：三分线哪段都一样，守哪儿无所谓。 （澄清：底角是最短的三分、期望分最高（≈ 1.1 分），且防守者扑防距离最短、轮转路线最远。现代防守铁律是「宁可放弧顶，也别漏底角」。）
  • 误解：把对手这次出手防到投丢，这个回合就赢了。 （澄清：还差最后一步——抢下防守篮板结束回合。漏了进攻篮板，对手白得一个额外回合，你前面压低 PPP 的努力被一笔勾销（第 10 课细讲）。）
一句话带走
防守是进攻算术的镜像：唯一目标是最小化对手每回合期望分（PPP）。把第 03 课那张期望分地图倒过来读，防守优先级自动浮现——① 死守篮下（最贵 1.2 分，护框第一）② 守住三分线尤其底角（第二贵 1.1 分）③ 主动把对手逼向中距离（最便宜 0.8 分，让他投那儿反而好）④ 绝不送罚球（1.54 分，全场最贵的得分，犯规＝负期望分）⑤ 不让进攻篮板（别白送额外回合）。把这五条归并去重，正好就是「防守版四要素」（限制 eFG%、逼失误、保护防守篮板、不送罚球，第 11 课展开）。这台「最小化对手期望」的引擎，足球（压缩高 xG 空间）、棒球（制造出局）通用。
下一步
防守的目标（压低对手 PPP）和优先级（护框 → 守三分 → 逼中距离 → 不犯规 → 保篮板）现在都清楚了。可这五条优先级，没有一条是一个人能独自完成的——护框的同时谁去守底角？逼突破的同时谁来补防漏掉的人？防守，是五个人的同步舞蹈。 → 第 09 课《防守体系：盯人、联防、换防与轮转》会把镜头从「目标」拉到「怎么用五个人协同实现它」：盯人 vs 联防，是「守人还是守空间」的根本取舍；换防（switch）一招消灭挡拆，却会制造错位（还记得第 04、07 课那块被换出来的「肥肉」吗）；而协防与轮转（rotation），是用一个人去补两个位、再依次补位的连锁反应——只要有一个人慢了半拍，整条链就会在某处漏出一个空位。
