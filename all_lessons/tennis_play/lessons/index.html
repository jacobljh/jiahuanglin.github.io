从零学会打网球

一条线性思维的推导线：网球的力不是从手臂来的，是从地面来的。身体是一条动力链（腿→髋→躯干→肩→臂），大肌肉先发力、一节节把速度传到手臂这根「鞭梢」，最后甩给球。于是每一个「正确动作」——握拍、站位、触球点、发球——都不是要你背的规定，而是从「这一下要干什么」反推出来的唯一顺手解。这门课带一个连怎么握拍都不知道的人，一步步把这条链接顺、真正把球打起来。

为什么是这门课
你使出全力，球却软趴趴、还老是飞出界——不是你力气不够，是你把最弱的手臂当成了发动机。这门课面向完全零基础的你：不谈战术、不谈看球门道，只教身体到底该怎么动。我们只立一个引擎——动力链（力从地面来）——然后按「先站稳 → 再发力 → 再打起来」的顺序，把握拍、平衡、正手、上旋、发球一个个推导出来。你不会背到一堆口诀，而会理解每个动作为什么非这样不可，于是练起来更快、更稳、更省力。
姊妹线：网球的逻辑
这门课是《网球的逻辑》的前传：那门课讲「为什么这么打」——嵌套计分、杠杆、发球权、数据革命；这门课讲更早的一步，「身体怎么动才能把球打好」。先在这里学会正确地打，再去那边看懂聪明地打。它也和「从零学会打」的其它姊妹线同源——篮球、足球、高尔夫等——它们共用同一台引擎：力从地面来，手只是末端的鞭梢；新手的通病都是「只甩鞭梢」。想把身体这台机器练得更强，可参看《训练的逻辑》。每课配一个浏览器里当场绘制的可交互小实验（无需任何图片或视频）。

第一部分 · 站稳（先能可复现地打到球）

00
导览：为什么力不是从手臂来的
用「你越使劲球越软」当钩子，立起全程引擎——动力链：力从地面起，经腿→髋→躯干→肩→臂传到球拍，手臂只是末端鞭梢。讲清这门课的方法论：不背动作，只从「任务」反推动作。一个滑块让你眼看着「只用手臂 vs 用上全身」把球速翻倍。
01
握拍与预备姿势：让动力链有一个听话的末端
力要一路传到球拍，第一个接口就是「手怎么握拍、身体怎么站着待命」。从任务反推：想给球加上旋（拍面上刷）→ 于是正手用东方 / 半西方式；想让发球截击拍面灵活 → 于是用大陆式。握法不是规定，是被触球任务逼出来的顺手解。
02
平衡、重心与用脚步先到位：给动力链一个稳的底座
动力链要从地面发动，就得先有个稳的底座。分腿垫步（split-step）、侧身、压低重心、用脚步把身体先送到球边——让「手」永远在一个舒服、平衡的位置完成击球。身体先于球到位，是一切可复现的前提。

第二部分 · 发力与控制（把球打快、打准）

03
正手抽球：把动力链一节节接顺
核心的一下。从地面反推：蹬地 → 转髋 → 转肩 → 挥臂 → 随挥，力一级级放大再甩给球。手臂最后才动、只负责把攒好的速度释放出去。点破新手最大的通病——「只用胳膊抡」，看它如何让球又软又累。
04
触球点与上旋：从「打到」升级到「打准、打稳」
力传到了，还得在对的位置、对的时机交给球。触球点在身体前方、腰腹高度；拍面由低到高刷过球背制造上旋（让球过网后往下扎，你才敢大力打不出界）；随挥的方向就是球的方向。控距离靠挥幅与旋转，不靠蛮力。
05
发球：把动力链竖起来用的另一块基石
发球是全场唯一你能完全掌控的一击，也是动力链的「竖版」：稳定的抛球 + 大陆式握拍 + 蹬地展体 + 手臂内旋（pronation），在最高触球点把力甩给球。抛球位置几乎决定一切。同样从任务反推，而不是硬背动作。

第三部分 · 打起来（真的开始玩）

06
常见错误的自查与纠正：每个坏习惯都是动力链断在某一环
把新手最常犯的六个错误——只用手臂、盯球没盯到底、握太紧、不随挥、站太直、不侧身——逐一还原成「动力链断在哪一节」，给出可自查的信号（怎么发现自己错了）和对症的纠正练习与口诀。
07
打你的第一场 + 继续进步的路线
热身与护身、从发球线开始的 mini-tennis、找人喂球对练、最基本的计分与球场礼仪；再给一张「刻意练习」路线图：先求落点稳定 → 再加力量 → 再上发球。会打了，就可以交棒《网球的逻辑》，去看懂你正在打的这项运动。

顺带一提：这条「从零学会打」的线和它的姊妹线——篮球、足球、棒球、高尔夫、冰球、美式橄榄球、开赛车——出自同一个家族：把「怎么把一项运动打起来」讲成一条从物理反推动作的推导线，而不是一堆要背的口诀。学会之后，再去各自的《X 的逻辑》看门道。
