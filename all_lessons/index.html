RL · Part 1 — Foundations

The theory prequel to the systems-side post-training course. A first-principles, linearized track from the Markov decision process to the frontiers. Part I (01–05): the value branch (Q-learning, DQN), the policy branch (policy gradient, Actor-Critic), planning (DP, MCTS), and exploration (bandits, UCB, Thompson). Part II (06–18): deep RL, the policy-gradient lineage (REINFORCE → GAE → importance sampling → TRPO → PPO), the LLM era (PPO/DPO/GRPO/RLHF, handing off to the post-training course), then the frontiers — imitation/inverse RL, continuous control (DDPG/TD3/SAC), and offline RL (BCQ, CQL). Part III (19–31): each application — recommendation, robotics, finance, scheduling, NLP, vision, platforms — mapped back to the core method it reuses. Closes with an interactive self-test.
